      நித்தி ஆயோக்

      நிதி ஆயோக் 2018ம் ஆண்டு இறுதிக்கான ஆய்வறிக்கை
      Posted On: 12 DEC 2018 1:43PM by PIB Chennai


    முன்முயற்சிகளும் திட்டங்களும்:


    (i) செயல்பாட்டை மதிப்பிடுதல் மற்றும் முக்கிய துறைகளில் செயல்பாடுகள் குறித்து மாநிலங்களின் தர வரிசை அமைத்தல்


    கல்வி, மக்கள் நல்வாழ்வு, குடிநீர், நீடித்த வளர்ச்சி இலக்கு ( SDG ) ஆகிய முக்கியமான துறைகளில் ஆண்டுதோறும் மேம்பட்ட விளைவுகளை மதிப்பீடு செய்வதற்கு நிதி ( NITI ) சில குறியீடுகளை இறுதி செய்து வகுத்துள்ளது. மருத்துவமனைகளின் செயல்பாடுகளை மதிப்பிடுவதற்கும் கண்காணிப்பதற்கும் மாவட்ட மருத்துவமனை குறியீடு ( District Hospital Index ) வகுக்கப்பட்டுள்ளது. இது தொடர்பாக வழிகாட்டிக் கையேடு 2016ம் ஆண்டு உலக சுகாதார தினத்தன்று ( World Health Day 2016 ) வெளியிடப்பட்டது. தற்போது மத்திய சுகாதாரம் மற்றும் குடும்ப நல அமைச்சகம் இந்திய புள்ளியியல் நிறுவனம் ( Indian Statistical Institute ) ஆகியவை இணைந்து புள்ளி விவரம் தயாரிக்கப்பட்டுள்ளது.


    (ii) மக்கள் வள மூலதனத்தை (SATH) மாற்றியமைப்பதற்கான நீடித்த செயல்பாடு


    மக்கள் வள மூலதனம் ( SATH ) என்பது மாநிலங்கள் தங்களது சமூகத் துறை குறிகாட்டிகளை ( Social Sector Indicators ) மேம்படுத்தும் வகையில் தொழில்நுட்ப ஆதரவை மூன்றாண்டுகளுக்கு அளித்து கல்வி மற்றும் சுகாதாரம் ஆகிய இரு முக்கிய துறைகளில் மாற்றத்தை ஏற்படுத்தும் முயற்சியாகும். தேர்ந்தெடுக்கப்பட்ட மாநிலங்களில் புதிய சவால்களைச் சமாளிக்கும் முறையில் இத்திட்டம் தொடங்கப்பட்டது. ஒவ்வொரு முன் முயற்சிக்கும் காலாண்டு மதிப்பீட்டுடன் மாநிலங்களில் மாற்றம் ஏற்படுத்துவதற்கான திட்ட உத்தி ( Roadmaps ) இறுதி செய்யப்பட்டுவிட்டது.


    (iii) ஒரே பாரதம் சிறந்த பாரதம் (Ek Bharat Shrestha Bharat)


    மாநிலங்களுக்கு இடையில் நீண்டகாலத்திற்கு கலாசார பரிவர்த்தனை, கல்விப் பரிமாற்றம் ஆகியவற்றை மேம்படுத்தி, நம் நாட்டை ஒற்றுமையுடனும் வலுவாக்குவதற்காக ஒரே பாரதம், சிறந்த பாரதம் ( EBSB ) என்ற கருத்தியல் உருவாக்கப்பட்டது. இது தொடர்பாக ஆறு மாநிலங்கள், யூனியன் பிரதேசங்கள், உயர்கல்வித் துறைகள், மனித ஆற்றல் மேம்பாட்டுத் துறை அமைச்சகம் ஆகியவை இடையில் புரிந்துணர்வு ஒப்பந்தங்கள் ( MoUs ) ஏற்படுத்தப்பட்டுள்ளன.


    ( iv) மாநிலங்களுக்குத் தேவையான கட்டமைப்பை மேற்படுத்தும் மேம்பாட்டு ஆதரவு சேவைகள் (DSSS)


    அனைத்து கட்டமைப்புத் துறைகளிலும் மத்திய மாநில உறவுக் கூட்டாளி மாதிரியை உருவாக்கவும், தனியார் அரசுத் துறை கூட்டாண்மை மீட்டுருவாக்கவும் மேம்பாட்டு ஆதரவு சேவைகள் (DSSS) ஏற்படுத்தப்பட்டன. இது ஆபத்தைக் குறைக்கவும் வளர்ச்சித் திட்டங்களில் அடிப்படைக் கட்டமைப்பு பிரச்சினைகளைத் தீர்க்கவும் உருவாக்கப்பட்டது. அத்துடன் கல்வி நிறுவன மற்றும் அமைப்பு ரீதியிலான திறன்களைக் கட்டமைக்கவும் தொடங்கப்பட்டன .


    (v) சுகாதாரத் துறையில் அரசு – தனியார் கூட்டு


    இதயவியல், புற்றுநோய் இயல், நுரையீரல் இயல் ஆகியவை தொடர்பான நோய்த் தடுப்பு, நோய் அறிதல் மற்றும் சிகிச்சை குறித்த அரசாங்கத்தின் சுகாதார இலக்கை எட்டுவதற்கு உதவும் வழிகாட்டு கட்டமைப்பு தயாரிக்கப்பட்டுள்ளது. இது மாவட்ட அளவிலான மருத்துவமனையில் செயல்படுத்தப்படும். அத்துடன், இரண்டாம் நிலை மூன்றாம் நிலை நகரங்களிலும் இது தொடர்பான கவனம் செலுத்தப்படும். இதில் தனியார் மற்றும் தன்னார்வத் தொண்டு நிறுவனங்களின் சேவைகளும் ஈடுபடுத்தப்படும்.


    (vi) மத்திய அமைச்சகங்களுடன் தொடர்புடைய நிலுவையில் மாநில பிரச்சினைகளைத் தீர்த்தல்


    அனைத்து மத்திய அமைச்சகங்களுடனும் ராஜஸ்தான், இமாசலப் பிரதேசம், பிகார், ஒடிசா, புதுச்சேரி ஆகிய மாநில யூனியன் பிரதேச அரசுகளின் நிலுவையில் உள்ள அனைத்துப் பிரச்சினைகளும் தீர்க்கப்பட்டுவிட்டன.


    (vii) மாநில மனித மேம்பாட்டு அறிக்கை


    மகாராஷ்டிரம், அசாம், தமிழ்நாடு, குஜராத், கர்நாடகம், தில்லி உள்ளிட்ட மாநிலங்கள், மத்திய இந்தியாவின் பந்தேல்கண்ட் ( Bundelkhand ) மண்டலம் ஆகியவை மாநில மனித மேம்பாட்டு அறிக்கையைத் தயாரிப்பதற்கு துணை புரியப்பட்டது.


    (viii) ஆர்வமுள்ள 115 மாவட்டங்கள் அடையாளம் காணுதல்


    ‘அனைவரும் இணைவோம், அனைவரும் உயர்வோம்’ ( SabkaSaath, SabkaVikas ) என்ற இலக்கை எட்டுவதற்காக அனைத்தும் உள்ளடக்கிய வளர்ச்சியைக் காணவும் பிரதம மந்திரி ஆர்வமுள்ள மாவட்ட திட்டத்தை ( Aspirational District Programme ) 2018, ஜனவரி 5ம் தேதி தொடங்கிவைத்தார். அதன் கீழ் 49 முக்கிய செயல்திறன் குறிகாட்டிகள் ( Key Performance Indicators ) அனைத்துப் பிரிவிலும் அடையாளம் காணப்பட்டுள்ளன. இது வாழ்க்கையை எளிதாக்கவும், அந்த மாவட்டங்களில் வாழும் குடிமக்களின் பொருளாதார உற்பத்தித் திறனை மேம்படுத்தவும் இது உதவும்.


    இந்த மாவட்டங்களில் நிதி ஆயோக் ( NITI Aayog ) 2018ம் ஆண்டு ஏப்ரல் 1ம் தேதி அடிப்படை தரமதிப்பீட்டு முறையை ( Baseline Ranking ) வெளியிட்டது. அதிலிருந்து மாவட்டங்கள் இத்துறைகளில் தங்களது நிலையை உறுதிப்படுத்திக் கொண்டு, மாநிலத்திலும், நாட்டிலும் சிறந்த மாவட்டமாகத் திகழ செயல்படலாம்.


    மேற்கண்ட குறிகாட்டிகளைப் பறைசாற்றுவதற்காக ‘ மாற்றங்களின் முதன்மையர் ’ ( Champions of Change ) என்ற பெயரில் ஆர்வமுள்ள மாவட்டங்களுக்காக தகவலேடு ( Dashboard ) வடிவமைக்கப்பட்டுள்ளது. அதில், புள்ளி விவரம், தர மதிப்பீடு ஆகியவை இடம்பெறும்.


    அரசின் ‘அனைவரும் இணைவோம், அனைவரும் உயர்வோம்’ ( SabkaSaath, SabkaVikas ) என்ற இலக்கை அடைவதற்கான உள்ளடக்கிய வளர்ச்சியை மேம்படுத்துதல்

      * அரசியல் சட்டத்தின் ஆறாவது அட்டவணையின் கீழ் வரும் பிரிவுகளுக்குப் போதிய நிதி கிடைக்காமல் உள்ள மாநிலங்களுக்கு வெளிப்படையாகவும் சரிசமமாகவும் நிதி கிடைப்பதற்காக மத்திய பஞ்சாயத்து ராஜ் அமைச்சகம் புதிய வழிகாட்டி நெறிகளை மத்திய பஞ்சாயத்து ராஜ்ய அமைச்சகம் வெளியிட்டுள்ளது.

    நீண்ட காலத் தொலைநோக்குடன் சான்றுகளின் அடிப்படையில் கொள்கை வகுத்தலும் உற்பத்தித் திறனை மேம்படுத்துதலும்


    (i) மூன்றாண்டு தேசிய செயல் திட்டம் மற்றும் 75வது சுதந்திர ஆண்டில் புதிய இந்தியாவுக்கான உத்தி


    2017-18 முதல் 2019-20 வரையிலான மூன்று ஆண்டுகளுக்கு செயல் திட்ட நிரலை நிதி ஆயோக் வகுத்துள்ளது. இந்த செயல்திட்ட நிரல் மாறிய இந்தியாவின் வளர்ச்சிக்கான உத்திக்கு துணை புரியும்.


    (ii) மத்திய அரசு நிறுவனங்களைச் சீர்திருத்துதல்


    பொதுத் துறை நிறுவனங்களிலிருந்து பங்குகளை நான்கு வகை உத்திகளில் திரும்பப் பெறுவது தொடர்பாக நிதி ஆயோக் பல்வேறு அமைச்சகங்களுடன் கலந்து ஆலோசித்து சில பரிந்துரைகளை அளித்துள்ளது. பங்குகளைத் திரும்பப் பெறுவதற்கு 30க்கும் மேற்பட்ட பொதுத் துறை நிறுவனங்களுக்கு பொருளாதார விவகாரத்துக்கான மத்திய அமைச்சரவை கொள்கை அளவில் ஒப்புதல் அளித்துள்ளது.


    பொதுத் துறையில் 74 செயல்படாத, நஷ்டத்தில் இயங்கும் நிறுவனங்கள் குறித்து அரசுக்கு அறிக்கை அளிக்கப்பட்டுள்ளது. அப்பரிந்துரைகள் தற்போது பரிசீலிக்கப்பட்டு வருகின்றன. இதுவரை 15 பொதுத் துறை நிறுவனங்களை மூடப்பட்டு வருகின்றன.


    (iii) சமச்சீரான மண்டல மேம்பாடு


      * சிறப்பு நிதி அளிப்பு: மண்டல அளவிலான மேம்பாட்டுக்காக ஒடிசா, பிகார், மேற்கு வங்கம் ஆகிய மாநிலங்களுக்கு பின்தங்கிய மண்டலத்துக்கான மானிய நிதி (Backward Region Grant Fund- BRGF) ) சிறப்புத் திட்டத்தின் கீழ் நிலுவையில் உள்ள நிதியை அளிப்பதற்கு நிதி ஆயோக் பரிந்துரைத்துள்ளது. இதற்கு 12வது ஐந்தாண்டு திட்டத்தில் ஒப்புதல் அளிக்கப்பட்டது. அத்துடன், அரசமைப்புச் சட்டத்தின் ஆறாவது அட்டவணையின் கீழ் குறிப்பிடப்பட்டுள்ள அசாம், மேகாலயா, மிஜோரம், திரிபுரா ஆகிய மாநிலங்களுக்கு ஒருமுறை ஒதுக்கப்படும் சிறப்பு நிதி அளிக்கவும் பரிந்துரைக்கப்பட்டுள்ளது.

      * வடகிழக்கு மேம்பாட்டுக்குத் துணைபுரிதல்: வடகிழக்கு மாநிலங்களிலும் இமயமலைப் பிரதேசங்களிலும் புதிய தொழில் கொள்கையைச் செயல்படுத்த ஆய்வு செய்யவும், திட்ட உத்தியைத் தெரிவிக்கவும் நிதி ஆயோக் முதன்மை செயல் அலுவலர் தலைமையில் ஒரு கமிட்டி அமைக்கப்பட்டுள்ளது.
      * வடகிழக்குக்கு நிதி ஆயோக்கின் தனி அமைப்பு (NITI Forum): நிதி ஆயோக் அமைப்பின் முதல் மண்டல அளவிலான அமைப்பு அமைக்கப்பட்டுள்ளது. அதில் வடகிழக்கு மாநிலங்களைச் சேர்ந்தவர்களும் மத்திய அமைச்சகங்கள், துறைகளைச் சேர்ந்தவர்களும் இடம்பெற்றுள்ளனர்.
      * தீவுகளின் ஒட்டுமொத்த வளர்ச்சிக்காக… : தீவுகளின் மேம்பாட்டுக்கான ஒட்டுமொத்த மேம்பாட்டு வழிமுறைகளை மேற்கொள்ள நிதி ஆயோக் அறிவுறுத்தப்பட்டுள்ளது. அவ்வாறு மேம்படுத்தப்படும் தீவுகள் நீடித்த வளர்ச்சிக்கான தனித்துவமான மாதிரி தீவுகளாக ஆக்கப்படும். முதல் கட்டமாக 10 தீவுகள் எடுத்துக் கொள்ளப்படுகின்றன.
      * தீவு மேம்பாட்டு முகமை (IDA): மத்திய உள்துறை அமைச்சரின் தலைமையில் செயல்படும் தீவுக்கான மேம்பாட்டு முகமை ( IDA ) 2017ம் ஆண்டு ஜூன் மாதம் தொடங்கப்பட்டது. அதில் நிதி ஆயோக் அமைப்பின் முதன்மைச் செயல் அலுவலர் அமைப்பாளராக இருப்பார்.
      * ஐநா மேம்பாட்டுத் திட்டம் (UNDP) புவிசார் தகவல் முறை (GIS) அடிப்படையிலான திட்டம்: அரசு சேவைகள் குறித்து திட்டம் வகுத்தல், நிர்வாகம், கண்காணிப்பு ஆகியவற்றுக்கு புவிசார் தகவல் முறையை ( Geographical Information System ) கடைப்பிடிப்பது குறித்து நிதி ஆயோக் ஆராய்ந்து வருகிறது. குஜராத்தில் உள்ள பாஸ்கராசார்யா விண்வெளிப் பயன்பாடு மற்றும் தகவல் நிறுவனம் ( Bhaskaracharya Institute for Space Applications and Geoinformatics ) வடிவமைத்த “கிராம வடிவமைப்பு மற்றும் தாலுகா திட்ட வரைபடம்” ( Village Profile and Talukka Planning Atlas ) என்ற புதுமையான மாதிரி அடிப்படையில் இந்தப் பணி மேற்கொள்ளப்படுகிறது.

    (iv) மக்கள் நல்வாழ்வு மற்றும் சத்துணவுத் துறையில் சீர்திருத்தங்கள்: மக்கள்நல்வாழ்வுத் துறையில் தீவிரமாக சீர்திருத்தங்களை மேற்கொள்ள நிதி ஆயோக் தொடங்கியுள்ளது.

      * ஹோமியோபதி தேசிய ஆணையம் ( NCH ) சட்டமுன்வடிவு (2017) மற்றும் இந்திய மருத்துவமுறை தேசிய ஆணையம் சட்ட முன்வடிவு ( National Commission for Indian Systems of Medicine Bill , 2017) ஆகியவை விரிவான விவாதங்களுக்குப் பிறகு இறுதி செய்யப்பட்டுவிட்டன.
      * சத்து மேம்பாட்டுக்கு தேசிய அளவிலான உத்தி: விரிவான விவாதங்களை அடுத்து தேசிய அளவில் சத்து மேம்பாட்டுக்கான உத்தியை நிதி ஆயோக் வகுத்துள்ளது.\
      * போஷன் அபியான் தொடக்கம்: அடுத்த மூன்று ஆண்டுகளில் தேசிய அளவில் (குழந்தைகளின்) சத்து மேம்பாட்டுக்கு போஷன் அபியான் திட்டம் தொடங்கப்பட்டுள்ளது.
      * மருந்துகள் துறையில் சீர்திருத்ததை முடுக்கிவிடுதல்: மருந்துகள், மருத்துவப் பொருட்கள் மலிவான விலையிலும் தரமான வகையிலும் கிடைப்பதை உறுதி செய்ய கொள்கை வகுப்பதற்கான கருதுகோளை நிதி ஆயோக் அளித்துள்ளது.
      * மருந்தியல் கல்வி மற்றும் ஆய்வுக்கான தேசிய நிறுவனம் ( NIPER ) – மருந்தியல் கல்வி மற்றும் ஆய்வுக்கான தேசிய நிறுவனத்தின் செயல்பாடு ஆராயப்பட்டது. அக்கல்வியை மேம்படுத்துவதற்கான வழி வகைகள், யோசனைகள் தெரிவிக்கப்பட்டன.

    (v) இந்திய மின்சக்தித் துறை முடுக்கிவிடுதல்

      * இந்திய மின்சக்தி பாதுகாப்பு நிலை (India Energy Security Scenario, 2047) என்ற திட்டம் 2015ம் ஆண்டு மாற்றியமைக்கப்பட்டது. அத்துடன், தேசிய உறுதிப்படுத்திய பங்களிப்புக்கான ( INDC ) நிர்ணயிக்கப்பட்டது. ஆந்திரப் பிரதேசம், குஜராத், அசாம் ஆகிய மாநிலங்களின் மாநில மின்சக்தி கணக்கீட்டு முறையை ( State Energy Calculator ) மேம்படுத்த நிதி ஆயோக் ஆதரவு அளிக்கிறது. ஆந்திரப்பிரதேசம் மாநில மின்சக்தி கணக்கீட்டு முறையை ( State Energy Calculator 2050 ) 2017ம் ஆண்டு நவம்பர் 16ம் தேதி தொடங்கியது. மற்ற இரு மாநிலங்களும் தங்களது முதல் வரைவுடன் தயாராக உள்ளன. இரண்டாம் கட்டமாக கர்நாடகம், தமிழ்நாடு, மகாராஷ்டிரம் ஆகிய மாநிலங்களில் எடுத்துக் கொள்ளப்படும்.

    குறுக்குவழி இடையீடுகள்

      * வேலைவாய்ப்பு மற்றும் ஏற்றுமதிக்கான பணிக்குழு: வேலைவாய்ப்பு மற்றும் ஏற்றுமதிகளுக்காக நிதி ஆயோக் துணைத் தலைவரின் தலைமையில் உயர்நிலை பணிக் குழுவை ( High Level Task Force ) நிதி ஆயோக் 2017, செப்ம்பரில் நிறுவியது. மத்திய அரசின் முதுநிலை செயலர் நிலையிலான அலுவலர்கள், வெளியுறவு நிபுணர்கள் இந்தப் பணிக்குழுவில் இடம்பெற்றுள்ளனர். இப்பணிக் குழு பல்வேறு துறைகளில் ஆய்வு நடத்தி, வேலைவாய்ப்புக்கும் ஏற்றுமதிகளுக்கு உரிய யோசனைகளை மத்திய வர்த்தகத் துறை அமைச்சரிடம் அளித்துள்ளது.
      * ஊரகப் பகுதிக்கான குடிநீர் – ஃப்ளோரைடு உள்ளிட்ட ரசாயனத்தால் பாதிக்கப்பட்ட 19 மாநிலங்களில் குடிநீரை மேம்படுத்துவதற்காக ரூ. 1000 கோடி அளவுக்கு நிதி அளிக்கப்பட்டுள்ளது. இதுவரை 14 மாநிலங்களில் 3100த்துக்கு மேற்பட்ட குடியிருப்புகளில் பாதுகாப்பான குடிநீர் வழங்க ஏற்பாடு செய்யப்பட்டுவிட்டது.
      * தூய்மை இந்தியா இயக்கம் (SBM) – தூய்மை இந்தியா இயக்கம் ( Swachh Bharat Mission ) 2014, அக்டோபர் 2ம் தேதி தொடங்கப்பட்டது. அது முதல் நாடு முழுவதும் 3 லட்சத்து 64 ஆயிரம் கிராமங்கள், 385 மாவட்டங்கள், 13 மாநிலங்கள், 4 யூனியன் பிரதேசங்களில் திறந்தவெளிக் கழிப்பிடம் முற்றிலுமாக ஒழிக்கப்பட்டுவிட்டது. கிராமப்புறங்களில் வீடுகளில் கழிவறைகள் கட்டுவது 39 சதவீதத்திலிருந்து 84 சதவீதமாக உயர்ந்துள்ளது.

    அமலாக்கத்தை மேம்படுத்துவதற்காகத் திட்டக் கண்காணிப்பு, அரசுத் திட்டங்களின் பலன்கள் ஆகியவற்றை அமைப்பு முறையாக்குதல்


    (i) விளைவுகளைக் கண்காணிக்கும் கட்டமைப்பு (2018-19):


    பயன்சார்ந்த கண்காணிப்பு முறைக்கு முக் கியத்துவம் தரும் வகையில் கண்காணிப்பு மற்றும் மதிப்பீட்டு அலுவலகம் ( Monitoring and Evaluation Office - DMEO ) சிறந்த வகையில் பலன்கள், வெளிப்பாடுகள் கிடைக்கும் வகையில் ஒரு வழிமுறையைக் கண்டறிந்துள்ளது. இது அனைத்து மாநிலத் தலைமைச் செயலகங்கள், மத்திய தலைமைச் செயலகம் ஆகியவற்றின் செலவினங்களுக்கான மதிப்பீட்டு குறிகாட்டிகளுடன் ( Measurable Indicators ) செயல்படும்.


    அனைத்து தலைமைச் செயலகங்கள், மத்திய தலைமைச் செயலகம் ஆகியவற்றின் நடப்பு ஆண்டுக்கான மொத்தம் ரூ. 8,14,00,000 கோடி மதிப்பிலான 600 வகையான செலவினங்களையும் கண்டறிவதற்கான செயல்பாடு தொடங்கிவிட்டது. இதற்கான வெளியீடுகள், பலன்கள், குறிகாட்டிகள் உலகத் தரமான நடைமுறையில் வடிவமைக்கப்பட்டுள்ளன.


    (ii) துறை ரீதியிலாக பிரதமர் அலுவலகம் மேற்கொள்ளும் ஆய்வுகள்:


    திட்டங்கள், துறைகளின் மூலம் பலன்கள் கிடைப்பதை அறிவதற்காக 15 துறைகளின் வெளியீடுகள் மதிப்பீட்டுக்காக எடுத்துக் கொள்ளப்படுகின்றன. ஆய்வுகள் பல ஆண்டுகளாக மேற்கொள்ளப்பட்டு வருகின்றன என்றாலும் கடந்த இரு ஆண்டுகளாக அனைத்து முன்னேற்றங்களும் கண்டறியப்பட்டு வருகின்றன.


    (iii) பலன்சார் நிதிநிலை


    மத்திய அரசின் பலன்சார் நிதிநிலை ( Union Outcome Budget ) மூலம் ஏற்படும் முன்னேற்றத்தை அவ்வப்போது பதிவேற்றுவதற்காக ஒரு தகவலேடு ( Dashboard ) உருவாக்கப்பட்டுள்ளது. அமைச்சகங்களும் துறைகளும் அதில் தங்களது விவரங்களைப் பதிவேற்றம் செய்யலாம்.


    (iv) திட்டக் கண்காணிப்பு மற்றும் மதிப்பீடு


    குறிப்பிட்ட திட்டங்களுக்கு கோரிக்கைகளின் அடிப்படையில் மதிப்பீட்டுப் பணிகளைக் கண்காணிப்பு மற்றும் மதிப்பீட்டு அலுவலகம் ( DMEO ) மேற்கொண்டுள்ளது. வெறும் செலவிடுதல் என்பதிலிருந்து பலன்சார்ந்த ஆளுகை என்ற நிலைக்கு முற்றிலும் புதிதாக மாறிய பிறகு, பலன்கள் பெறப்பட்டனவா என்பதில் கவனம் செலுத்தப்படுகிறது. சாதனைகளும் அலசப்படுகின்றன.


    இது விஷயத்தில், இணையதளம் சார்ந்த பரிமாற்றத் தகவலேடு ( Web-Based Interactive Dashboard ) கண்காணிப்பு மற்றும் மதிப்பீட்டு அலுவலகத்தால் ( DMEO ) உருவாக்கப்பட்டுள்ளது. இது பிரதம மந்திரி வீட்டுவசதித் திட்டத்தின் ( Pradhan MantriAawasYojana ) கீழ் கிராமங்கள், நகரங்களில் வீடுகள் கட்டுவதற்கான பணியின் முன்னேற்றம் கண்காணிக்கப்படும். இந்தத் தகவலேட்டின் திரைப்படிவம் கீழே தரப்பட்டுள்ளது:


    மேலும், தற்போது செயலாக்கப்பட்டுவரும் குறிப்பிட்ட சில திட்டங்களின் மீதான மதிப்பீடு எடுத்துக் கொள்ளப்படும். திட்டங்களில் இடைவெளி இருப்பதையும் கண்டறியவும் விரைந்து ஆய்வு செய்யப்படும்.


    (v) நீடித்த வளர்ச்சிக்கான இலக்குகள் குறித்து செயலாக்கம் மற்றும் கண்காணிப்பில் ஏற்படும் முன்னேற்றம்


    (a) உலகளாவிய நீடித்த வளர்ச்சி இலக்குக்கான ( Sustainable Development Goals - SDGs ) குறிகாட்டிகளுக்கு ஐ.நா. புள்ளியியல் ஆணையம் ( UN Statistical Commission ) அளித்த ஒப்புதலின்படி, தேசிய அளவிலான நீடித்த வளர்ச்சி இலக்கு ( SDG ) பட்டியலை மத்திய புள்ளியில் மற்றும் திட்ட செயலாக்க அமைச்சகம் தயாரித்துள்ளது. அதன் செயலாக்கத்தைக் கண்காணிக்கும் பணி நிதி ஆயோக் ( NITI Aayog ) அமைப்பிடம் அளிக்கப்பட்டுள்ளது. நீடித்த வளர்ச்சி இலக்கு குறித்த பணிக்குழுவை நிதி ஆயோக் அமைத்துள்ளது. அதன் செயலாக்கத்தை மத்திய அமைச்சகங்கள், மாநில அரசுகள், சிந்தனையாளர்கள் கண்காணிப்பர்.


    (vi) மத்திய அரசின் திட்ட மதிப்பீட்டுப் பிரிவு:


    நிதி ஆயோக் அமைப்பு 2015, ஜனவரி 1ம் தேதி முதல் மொத்தம் ரூ. 45,14,389 மதிப்பீட்டில் அரசு நிதியிலான 584 திட்டங்களை ஆய்வு செய்துள்ளது. அது தவிர, ரூ. 2,16,703 மதிப்பில் அரசு – தனியார் பங்களிப்பிலான 277 திட்டங்கள் ஆய்வு செய்யப்படுகின்றன. இவற்றில் 229 மத்திய அரசின் திட்டங்களும், 48 மாநில அரசுகளின் திட்டங்களும் ஆகும்.


    (vii) யூனியன் பிரதேசங்களின் செயல்பாடுகள் குறித்த மதிப்பீடு:


    யூனியன் பிரதேசங்களின் வளர்ச்சி, முன்னேற்றத்தைத் தொடர்ந்து கண்காணிக்க தகவலேட்டை ( Dashboard ) நிதி ஆயோக் (NITI Aayog ) வடிவமைத்துள்ளது. அந்த அரசுகளின் திட்டங்கள் செயல்படுத்தப்படுவது குறித்து தொடர் ஆய்வு மேற்கொள்ளப்படும்.


    தேசிய, சர்வதேச அளவிலான கூட்டு மற்றும் கொள்கை வகுப்பதில் அவற்றை ஈடுபடுத்துதல்


    NITI has provided the platform to collaborate with national and international think tanks on wide-ranging areas. பல்வேறு துறைகளில் தேசிய, சர்வதேச அளவிலான சிந்தனைக் குழுக்களுடன் இணைந்து செயல்படுவதற்கான தளத்தை நிதி ஆயோக் உருவாக்கியுள்ளது. மாநாடுகள், பயிலரங்குகள், கூட்டு ஆய்வுத் திட்டங்கள் ஆகியவற்றின் மூலம் கிடைக்கும் அறிவுசார் கருத்துகளை அரசின் கொள்கைகளை வகுப்பதற்குப் பயன்படுத்த நிதி ஆயோக் வழியமைத்துள்ளது.

      * நீதித்துறை நடைமுறையின் முக்கியமான விஷயங்களில் ஒருமித்த கருத்தை உருவாக்குவதற்கான மாநாடு-

    நாட்டில் வழக்குகளைத் தீர்ப்பதற்கான முயற்சி குறித்து விவாதிக்க உலகளாவிய மாநாடு ஏற்பாடு செய்யப்பட்டது. உலகின் மிகப் பெரிய கவுரம் மிக்க விருதான உலகளாவிய மத்தியஸ்த ஆய்வு ( Global Arbitration Review- GAR ) விருது இந்தியாவுக்குக் கிடைத்துள்ளது. “ பெரும் முன்னேற்றம் அடைந்த அதிகார எல்லை” ( Jurisdiction that has made great progress ) என்ற பிரிவுக்காக இந்த விருது கிடைத்துள்ளது.


    அறிவு மற்றும் புதுமையாக்க மையம்


    நல்ல ஆளுகைக்கான ஆய்வுத் தகவல் தொகுப்போ, நீடித்த மற்றும் சமமான வளர்ச்சிக்கான சிறந்த நடைமுறைகள், அறிவுசார் கருத்துகளைப் பகிர்ந்தளித்தல் ஆகியவற்றில் அதி உயர் திறன் கொண்ட வள மையமாகச் செயல்படுவது நிதி ஆயோக்கின் பல கடமைகளில் முக்கியமான ஒன்றாகும். அதன் அடிப்படையில் நல்ல ஆய்வுத் தகவல் தொகுப்பை உருவக்கும் முயற்சிகள் மேற்கொள்ளப்பட்டு வருகின்றன.


    தொழில்முனைவுக்கு உகந்த சூழலை மேம்படுத்துதல்

     1. அடல் புதுமையாக்க இயக்கம்
      * அடல் புதுமையாக்க இயக்கம் ( Atal Innovation Mission ) நிதி ஆயோக் அமைப்பின் கீழ் அமைந்த இந்திய அரசின் முதன்மையான முன்முயற்சியாகும். இது நாட்டில் புதுமையாக்கத்தை மேம்படுத்தவும் தொழில்முனைவை ஊக்குவிக்கவும் அமைக்கப்பட்டது. இதன் முதல் கட்ட நிகழ்ச்சிகளில் கல்லூரி, கல்லூரிகள் நிலையில் புதுமையாக்கத்துக்கான இணைப்பை ( Network Of Innovation ) அமைக்கப்பட்டுள்ளது.

    அடல் செம்மையாக்க ஆய்வகங்கள் ( ATLs )


    உயர்கல்வி நிறுவனங்களின் நிலையில், முதன்மையான திட்டமாக அடல் செம்மையாக்க ஆய்வகங்கள் (Atal Tinkering Labs - ATLs ) அமைக்கப்பட்டுள்ளன. அதன் கீழ் அடல் புதுமையாக்க இயக்கம் ( AIM ) 2,400க்கும் மேற்பட்ட அடல் செம்மையாக்க ஆய்வக பள்ளிகள் ( ATL Schools ) நாடு முழுவதும் அமைக்கப்பட்டுள்ளன. இந்த ஆய்வகங்களில் மூன்று லட்சத்துக்கும் மேற்பட்ட மாணவர்கள் 3,500 படைப்புகளை உருவாக்கியுள்ளனர். பல்வேறு வகையான செயல்பாடுகள் மூலம் 1000 ஆசிரியர்கள் பயிற்சி பெற்றனர். இத்திட்டத்தின் கீழ் 6வது வகுப்பு மாணவர்கள் முதல் ரோபாட்கள், முப்பரிமாண பிரிண்ட்கள், இணையம் ஆகியவற்றைக் கையாள்வர்.


    (ii) தொழில்முனைவுக்கான உலகளாவிய உச்சி மாநாடு 2017: மகளிர்க்கு முதலிடம், அனைவருக்கும் வளம்

      * நிதி ஆயோக் கடந்த 2017 நவம்பரில் ஹைதராபாதில் தொழில்முனைவுக்கான உலகளாவிய உச்சி மாநாட்டுக்கு ஏற்பாடு செய்தது. அம்மாநாட்டை பிரதமரும் அமெரிக்க அதிபரின் ஆலோசகர் திருமதி இவாங்க்கா டிரம்ப் ஆகியோர் தொடங்கிவைத்தனர். இந்த உச்சி மாநாடு திறமையான உலகின் முன்னணி தொழில்முனைவோரையும் முதலீட்டாளர்களையும் இணைக்க உதவியது. உலகளவில் உருவாகும் சவால்களை எதிர்கொள்ள புதிய தீர்வுகளைக் கண்டறியவும் உதவியது.

    (iii) மகளிர் தொழில்முனைவுத் தளம் Entrepreneurship Platform


    மகளிர்க்கான தொழில்முனைவு தளத்தை ( Women Entrepreneurship Platform ) நிதி ஆயோக் உலக மகளிர் தினத்தன்று தொடங்கியது. நாடு முழுவதும் பெண்கள் தொழில் தொடங்குவதற்கு உகந்த சூழலை ஏற்படுத்துவதே இதன் நோக்கமாகும். அத்துடன் பெண்கள் தங்களிடம் உள்ள தொழில்முனைவு ஆர்வத்தை உணர்ந்து புதிதாக தொழிலைத் தொடங்குவதற்கு புதிய வழிகளைக் கண்டறியவும் நீடித்த வகையிலான திட்டத்தை வகுக்கவும் நீண்ட கால உத்திகளைக் கையாளவும் வழியமைத்துள்ளது. இதற்காக http:// wep.gov.in என்ற இணையவாசல் தொடங்கப்பட்டுள்ளது. இது தகவல் அளிப்பதுடன் கருத்துப் பரிமாற்ற இணையமாகும். அறிவுசார் தளமாகவும் உள்ளது.


    வேளாண்மையில் வினையூக்க சீர்திருத்தம்


    விவசாயிகளின் வருமானத்தை 2022ம் ஆண்டுக்குள் இரு மடங்காக உயர்த்துவது என்ற அரசின் குறிக்கோளின்படி பல்வேறு தொடர் முன்முயற்சிகளில் நிதி ஆயோக் ஈடுபட்டுள்ளது. அதன்படி பல்வேறு சீர்திருத்த நடவடிக்கைகளைக் கொண்டு வருகிறது.


    நவீன தொழில்நுட்பத்தைக் கையாளுவதை ஊக்குவித்தல்


    (i) டிஜிட்டல் இந்தியா


    பண மதிப்பிழப்பிற்குப் பிறகு, டிஜிட்டல் முறையில் பணப் பரிமாற்றத்துக்கு நிதி ஆயோக் முக்கிய காரணியாக இருக்கிறது. டிஜிட்டல் பணப் பரிமாற்றத்தை பரவலாக்குவதற்காக நிதி ஆயோக் நீண்ட ஆலோசனையும் விரிவான ஆய்வும் நடத்தியுள்ளது. அனைத்து அமைச்சகங்கள், தொழில்நிறுவனங்கள், கல்வி நிறுவனங்களில் மாநிலங்களில் இது தொடர்பாக விழிப்புணர்வு செயல்களும் மேற்கொள்ளப்பட்டன.


    டிஜிட்டல் பணப் பட்டுவாடாவை பரவலாக்குவது தொடர்பான வழியைக் கண்டறிய ஆந்திரப் பிரதேச முதலமைச்சரை அமைப்பாளராகக் கொண்டு முதலமைச்சர்களைக் கொண்ட ஒரு கமிட்டியை நிதி ஆயோக் அமைத்துள்ளது.


    டிஜிட்டல் பணப் பரிமாற்றத்துக்கு உதவும் வகையில் பீம் ( Bharat Interface for Money ) எனப்படும் செயலியை நிதி ஆயோக் உருவாக்கி பயன்பாட்டுக்கு வெளியிட்டுள்ளது. இது தொலை தூரம் இருப்போருக்கு இடையில் பணப் பரிமாற்றத்தை எளிதில் மேற்கொள்ள பெரிதும் உதவும்.


    குறைந்த பணப் புழக்க நகரியம் என்ற திட்டத்தையும் நிதி ஆயோக் கொண்டு வந்துள்ளது.


    (ii) எளிதான போக்குவரத்து இணைப்பு வழிகள்


    யாரும் எங்கும் செல்வதற்கு வசதியாக, பல இடங்களை இணைக்கும் வகையிலான எளிதான போக்குவரத்து வழிகள் நாட்டில் பொலிவுறு நகரங்களில் கொண்டுவருவதற்கு நிதி ஆயோக் தீவிரமான முயற்சியை மேற்கொண்டு வருகிறது.


    மின்சாரத்தில் இயங்கும் வாகனங்களுக்குத் தேவையான மின் இணைப்பை வழங்கும் நிலையங்கள் ( EV Charging Stations ) நிதி ஆயோக் நிறுவனத்தில் அமைக்கப்பட்டுள்ளன.


    ஆறு அமைச்சகங்களுக்கு இடையிலான கமிட்டிகளை நிதி ஆயோக் கடந்த 2018 பிப்ரவரியில் அமைத்தது. இந்த கமிட்டிகள் தகவல் பரிமாற்றம் தொடர்பான உத்திகளில் பல்வேறு விஷயங்கள் குறித்து முடிவு செய்யும்.


    போக்குவரத்து குறித்த உலகளாவிய உச்சி மாநாடு 2018 செப்டம்பரில் நடைபெற்றது. போக்குவரத்து குறித்த பல்வேறு விஷயங்கள் குறித்து விழிப்பு ஏற்படுத்தவும், போக்குவரத்தை மேம்படுத்துவதில் சம்பந்தப்பட்டவர்களை ஈடுபடுத்துவதும் இதன் நோக்கமாகும்.


    (iii) ஆளுகையில் அதி தொழில்நுட்பங்களைக் கடைப்பிடித்தல்


    செயற்கை அறிவு: செயற்கை அறிவு (Artificial Intelligence - AI) என அழைக்கப்படும் கணினிசார் அறிவுத் திறன் தொடர்பான தேசிய திட்டத்தை உருவாக்கி செயல்படுத்தவும், இந்தியப் பொருளாதாரம், சமூகம், ஆளுகை ஆகியவற்றில் அதன் தாக்கம் குறித்து அறியவும் நிதி ஆயோக் நிறுவனம் அறிவுறுத்தப்பட்டுள்ளது. இத்தொழில்நுட்பத்தின் மேம்பாடு, கடைப்பிடிக்கும் நுட்பம், தொழில்நுட்பத் தாக்கம் ஆகியவற்றை அறிந்துகொள்ள பல்வேறு அமைச்சகங்கள், கல்வியாளர்கள், தொழில்நிறுவனங்கள், ஆய்வாளர்களுடன் கலந்து ஆலோசிக்கப்படுகிறது. அதன்படி செயற்கை அறிவுக்கான தேசிய உத்தி குறித்த வரைவு விவாத அறிக்கை தயாரிக்கப்பட்டு வருகிறது.


    தொடரேடு ( Blockchain ):


    தேசிய அளவில் அனைத்தும் இணைக்கும் இந்தியச் சங்கிலி ( IndiaChain ) குறித்த விவாத அறிக்கையை நிதி ஆயோக் தயாரித்து வருகிறது. ( தொடரேடு ( Blockchain ) என்பது அனைத்துத் தகவலும் ஒரே ஏட்டில் தொடர்ந்து பதியப்படும் தொழில்நுட்பமாகும்). இந்தியச் சங்கிலி என்பது ஆதார் ( Aadhaar ), ஒருங்கிணைந்த வழங்கீட்டு பொதுமுகம் ( UPI ), மின்கையெழுத்து ( eSign ) போன்ற இந்தியா முழுமைக்குமான கட்டமைப்பு இணைப்புக்கான ஒரு திட்டமாகும். நிதி ஆயோக்கின் அறிக்கையில் இந்தியச் சங்கிலியின் கருத்தியல் கட்டமைப்பு, வடிவமைப்பு ஆகியவை விவரிக்கப்படும்.


    (iv) மெத்தனால் பயன்பாடு : பெட்ரோலியம் உள்ளிட்ட கச்சா எண்ணெயைச் சார்ந்திருப்பதைக் குறைக்கவும், இறக்குமதியைக் குறைக்கவும் உதவும் திட்ட உத்தி குறித்து ஓர் உயர் கமிட்டியும் ஐந்து பணிக்குழுக்களும் தீவிரமாக ஆராய்ந்து வருகின்றன. நிலக்கரிச் சாம்பல், நகர திடக் கழிவுகள் ஆகியவற்றிலிருந்து மெத்தனால் உற்பத்தி செய்வது, மெத்தானல் இருப்பு வைத்தல், எடுத்துச் செல்லுதல், மெத்தனால் என்ஜின்கள் குறித்த ஆய்வு மேம்பாட்டுப் பணி ஆகியவை மேற்கொள்ளப்படுகின்றன.


    (v) பாதுகாப்புக் கவசங்களை இந்தியாவில் தயாரிப்பதற்கான உத்தி: இந்தியாவிலேயே பாதுகாப்புக் கவசங்களைத் தயாரிப்பது தொடர்பான உத்தியை நிதி ஆயோக் கமிட்டி ஒன்று தயாரித்து, பிரதமர் அலுவலகத்தில் அளித்துள்ளது. அதில் உள்நாட்டிலேயே பாதுகாப்புக் கவசங்களையும் அதற்கான மூலப் பொருட்களையும் தயாரிப்பதை மேம்படுத்துதல், கூடுதல் ஆய்வுக்கான வசதிகளை ஏற்படுத்துதல், கவசங்களுக்கு இந்தியாவின் தரநிர்ணய முறையைக் கடைப்பிடித்தல், எடைகுறைந்த கவசங்களைத் தயாரிக்க நேனோ தொழில்நுட்பத்தில் ஆய்வுக்கும் மேம்பாட்டுக்கும் ஏற்ற மேம்பாட்டு மையங்களை உருவாக்குதல், கொள்முதல் நடைமுறையை எளிதாக்குதல் ஆகிய முக்கியமான பரிந்துரைகள் இடம்பெற்றுள்ளன.


    சர்வதேச தொடர்புகள்

     1. சீனாவுடன் இந்தியச் சங்கிலியின் உத்திசார் பொருளாதாரம் குறித்த நான்காவது விவாதத்துக்கு இந்தியா ஏற்பாடு செய்திருந்தது. அதற்கு நிதி ஆயோக் துணைபுரிந்தது. 2016 அக்டோபர் 6, 7 ஆகிய நாட்களில் நடைபெற்ற இந்த விவாதத்தில் சீனாவின் சீர்திருத்த ஆணையம் பங்கேற்றது.
     1. அமெரிக்காவின் மின்சக்தி தகவல் அமைச்சகம் ( Energy Information Administration ) இந்திய எரிசக்தி அமைச்சகம் மற்றும் துறைகள் ஆகியவற்றுக்கு இடையிலான மின்சக்தி தகவல் மேலாண்மை குறித்த கூட்டங்கள் புது தில்லியில் 2016, அக்டோபர் 24ம் தேதி முதல் 27ம் தேதி வரையில் நடைபெற்றன.
     1. நிதி ஆயோக் அமைப்புக்கும் சீனாவின் மேம்பாட்டு ஆய்வு மையத்துக்கும் ( Development Research Centre ) இடையில் தில்லியில் 2016 நவம்பரில் இரண்டாவது கூட்டமும் பெய்ஜிங்கில் 2017 டிசம்பரில் மூன்றாவது கூட்டமும் நடைபெற்றன. நான்காவது கூட்டம் 2018 நவம்பரில் மும்பையில் நடைபெற்றது.
