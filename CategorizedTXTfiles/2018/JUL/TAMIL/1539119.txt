      சிறுபான்மையினர் நலன் அமைச்சகம்

      சிறுபான்மையினருக்கான நலத் திட்டம்
      Posted On: 18 JUL 2018 2:07PM by PIB Chennai


    மத்திய அரசு சமூகத்தில் சிறுபான்மை சமுதாயத்தினரின் நலனுக்காகப் பல்வேறு திட்டங்களை தொடங்கி கடந்த நான்காண்டுகளாக மேற்கொண்டு வருகிறது.


    “நயி மன்ஸில்” ( Nai Manzil ) என்ற திட்டம் பள்ளிப் படைப்பில் இடைநின்றவர்களுக்கு மரபுசார் கல்வியியையும், திறன்களை வளர்க்கும் பயிற்சிகளையும் அளிப்பதற்காக தொடங்கப்பட்டது. நயி மன்ஸில் என்றால் புதிய இலக்கு என்று பொருளாகும்.


    பாரம்பரிய கலைகள், கைவினைக் கலைகளில் சிறுபான்மையினர் தங்களது திறன்களை மேம்படுத்துவதற்காக “உஸ்தாத்” ( USTTAD ) என்ற திட்டம் கொண்டுவரப்பட்டுள்ளது.


    “ கைவினைஞர்கள் அங்காடி” என்ற பொருளில் அமைந்த ஹுனார் ஹாட் ( Hunar Haat ) திட்டம் 2016-17ம் ஆண்டு முதல் செயல்படுத்தப்பட்டு வருகிறது. பாரம்பரிய கைவினைக் கலைகள், கலைகள், வேலைவாய்ப்புக்குரிய பயிற்சிகளுடன் சந்தைக்கான தொடர்புகளையும் வலுப்படுத்துவதற்கான இத்திட்டம் 2017-18 ஆம் ஆண்டு வரையில் வெற்றிகரமாகச் செயல்படுத்தப்பட்டது. 2018-19ம் ஆண்டும் ஹுனார் ஹாட் இயக்கம் திட்டமிடப்பட்டுள்ளது.


    ஒட்டுமொத்த இந்திய பண்பாட்டின் ஓர் அங்கமான ஹமாரி தரோஹர் ( Hamari Dharohar ) என்ற திட்டம் சிறுபான்மையினரின் வளமான பாரம்பரியத்தைப் போற்றிப் பாதுகாப்பதற்கானது. கரீப் நவாஸ் ( Gharib Nawaz ) என்பது சிறுபான்மையினரின் திறன் மேம்பாட்டுக்காக மவுலானா ஆசாத் கல்வி நிறுவனத்தின் ( Maulana Azad Education Foundation ) கீழ் மேற்கொள்ளப்படும் பயிற்சித் திட்டமாகும். இது இளைஞர்களுக்கான குறுகிய கால வேலைவாய்ப்புப் பயிற்சித் திட்டமாகும். குறிப்பாக, மத்திய அரசால் அறிவிக்கப்பட்டுள்ள சிறுபான்மையினராகிய முஸ்லிம்கள், கிறிஸ்தவர்கள், சீக்கியர்கள், பவுத்தர்கள், பார்சிக்கள், ஜைனர்கள் ஆகியோருக்காகச் செயல்படுத்தப்படும்.


    மதரஸா பள்ளி மாணவர்களுக்காகவும் பள்ளிப் படிப்பையில் இடையில் கைவிட்டவர்களுக்காகவும் இணைப்புக் கல்விப் பயிற்சியை அமைச்சகம் தொடங்கியுள்ளது. நயி மன்ஸில் திட்டத்தின் கீழ் மவுலானா ஆசாத் கல்வி நிறுவனத்தின் ( MAEF ) ஆதரவில் இப்பயிற்சி அலிகர் முஸ்லிம் பல்கலைக்கழகம், புது தில்லியில் உள்ள ஜாமியா மிலியா இஸ்லாமியா ஆகியவை மூலம் நடத்தப்படுகிறது. அதைப் போல் ஸ்வச் வித்யாலயா திட்டமும் மவுலானா ஆசாத் கல்வி நிறுவனத்தின் ( MAEF ) மூலம் நடத்தப்படுகிறது.


    இத்திட்டங்கள் குறித்த விவரங்களை சிறுபான்மை அமைச்சகத்தின் www.minorityaffairs.gov.in என்ற இணையம் மற்றும் மவுலானா ஆஸாத் கல்வி நிறுவனத்தின் www.maef.nic.in என்ற இணையம் மூலம் அறிந்து கொள்ளலாம். அத்திட்டங்களுக்காக ஒதுக்கப்படும் நிதி, 2018, ஜூன் 30ம் தேதி வரையில் கடந்த மூன்று ஆண்டுகளில் பயன்படுத்தப்பட்ட நிதியளவு ஆகிய விவரங்கள் இந்த இணையதளங்களில் இடம்பெற்றுள்ளன.


    மேற்கண்ட தகவல்களை மத்திய சிறுபான்மையினர் நலத் துறை அமைச்சர் திரு. முக்தார் அப்பாஸ் நக்வி மக்களவையில் இன்று எழுத்துமூலம் அளித்த பதிலில் தெரிவித்துள்ளார்.
