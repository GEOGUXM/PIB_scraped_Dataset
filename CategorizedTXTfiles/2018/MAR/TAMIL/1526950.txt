      விண்வெளித்துறை

      சந்திரனுக்கும் செவ்வாய்க்கும் விண்வெளி பயணம்
      Posted On: 28 MAR 2018 3:45PM by PIB Chennai


    சந்திரனுக்குப் பயணம் செய்யும் “சந்திரயான் – 2” முழுக்க முழுக்க உள்நாட்டிலேயே வடிவமைக்கப்பட்டு, செயல்படுத்தப்படும் பயணம் ஆகும். அந்த ராக்கெட்டில் சுற்றுப் பாதையில் பயணம் செய்யும் பிரிவு ( Orbiter ), தரையிறங்கும் பிரிவு ( Lander ), நிலாவின் தரையில் நடமாடும் பிரிவு ( R over ) ஆகிய மூன்று முக்கியப் பிரிவுகள் உள்ளன. இவற்றில் சுற்றுப்பாதைப் பயணி ( Orbiter ) நிலாவிற்கு 100 கி.மீ. தொலைவில் சுற்றுப் பாதையில் நிறுத்தப்படும். “சந்திரயான்-2” விண்கலம் நிலாவுக்கு 100 கி.மீ. தொலைவை அடைந்ததும் சுற்றுப்பாதைப் பிரிவு ( Orbiter ) பிரிக்கப்படும். அதையடுத்து, நிலவில் இறங்கும் பிரிவு ( Lander ) பத்திரமாக, நிதானமாகத் தரையிறங்கும். அதையடுத்து, அதிலிருந்து நடமாடும் பிரிவு ( R over ) தனியாகப் பிரிந்து நிலவின் தரையில் நடமாடி ஆய்வு மேற்கொள்ளும். அப்போது, சுற்றுப்பாதைப் பிரிவு ( Orbiter ) தொடர்ந்து நிலவைச் சுற்றிப் பறந்துகொண்டிருக்கும். அங்கிருந்து தொலையுணர்வு மூலம் நிலவின் நிலப் பகுதிகளைக் கண்டறிந்து தெரிவிக்கும்.


    இந்திய விண்வெளி ஆய்வு மையம் ( ISRO ) சூரியக் குடும்பம் குறித்த ஆய்வுகளை மேற்கொள்வது தொடர்பாக ஓர் ஆய்வுக் குழுவை அமைத்துள்ளது. அந்த ஆய்வுக் குழு எதிர்காலத்தில் மேற்கொள்ளும் செவ்வாய்க் கிரகப் பயணம் குறித்து பரிந்துரைகளை அளித்துள்ளது. இது தொடர்பான தரவுகளுக்கான அறிவியல் முன்மொழிவுகளை வல்லுநர் குழு தேர்ந்தெடுத்துள்ளது.


    இத்தகவல்களை வடகிழக்கு மண்டல மேம்பாடு (தனிப்பொறுப்பு), பிரதமர் அலுவலகம், பணியாளர் நலன், பொதுமக்கள் குறைதீர்ப்பு மற்றும் ஓய்வூதியம், அணுசக்தி, விண்வெளித் துறை இணையமைச்சர் டாக்டர் ஜிதேந்திர சிங் நாடாளுமன்ற மக்களவையில் இன்று கேள்வி ஒன்றிற்கு எழுத்துமூலம் அளித்த பதிலில் தெரிவித்துள்ளார்.
