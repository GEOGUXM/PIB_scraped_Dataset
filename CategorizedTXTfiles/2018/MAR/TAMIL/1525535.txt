      பெண்கள் மற்றும் குழந்தை நலன் அமைச்சகம்

      சத்துக் குறைபாட்டுக்குத் தீர்வுகாண ஊட்டச் சத்து குறித்த (போஷன்) பிரசாரம்
      Posted On: 20 MAR 2018 7:34PM by PIB Chennai


    ஊட்டச் சத்துக் குறைபாட்டைத் தீர்ப்பது ( POSHAN Abhiyaan ) குறித்த தேசிய அளவிலான பயிலரங்கத்தை மத்திய மகளிர் நலன் மற்றும் குழந்தைகள் மேம்பாட்டு அமைச்சகம் புது தில்லி, ப்ரவாசி பாரதீய கேந்திரத்தில் இன்று (மார்ச் 20) நடத்தியது. ஒரு நாள் முழுவதும் நடைபெற்ற இந்தப் பயிலரங்கில் அனைத்து மாநிலங்களும், யூனியன் பிரதேசங்களும் பங்கேற்றன.


    இந்தப் பயிலரங்கை ஒட்டி மின்வழி வளர்கல்வி அணுகுமுறை ( e-Incremental Learning Approach ) குறித்த ஐந்து வகையான படிப்புகளும், குழந்தைப் பருவ நலன் மற்றும் கல்வி ( Early Childhood Care and Education ) சார்ந்த இரு வகையான சாஃப்ட்வேர் தொகுப்புகளும் ( Software modules ) தொடங்கப்பட்டன.


    இவற்றைத் தொடங்கிவைத்துப் பேசிய மகளிர் மற்றும் குழந்தைகள் மேம்பாட்டுத் துறையின் செயலர் திரு. ராகேஷ் ஸ்ரீவஸ்தவா, “ஊட்டச்சத்துப் பிரசாரம் ( POSHAN Abhiyaan ) திட்டத்திற்காக மூன்றாண்டு நிதியாக ரூ. 9046.15 கோடியை மத்திய அரசு 2017-18 ம் நிதியாண்டிலிருந்து ஒதுக்கியுள்ளது. மின்வழி வளர்கல்வி அணுகுமுறை ( e-ILA ) ஆன்லைன் வழிக் கல்வியாகும். இதில், ஒவ்வொரு சவாலையும் சரியாகவும் தொடர்ச்சியாகவும் திட்டமிட்டு எதிர்கொண்டு செயல்படுவதற்கான நிரல் இயக்கிகள் ( P rogramme functionaries ) தற்போதைய திறன் மேம்பாட்டின் அடிப்படையில் மிகவும் பயனுள்ளதாக அமைந்திருக்கும்” என்று குறிப்பிட்டார்.


    குழந்தைப் பருவ நலன் மற்றும் கல்வி ( ECCE ) சார்ந்த சாஃப்ட்வேர் தொகுப்புகளும் ( Software modules ) அங்கன்வாடிப் பணியாளர்களுக்குப் பயிற்சி அளிப்போருக்கு வழிகாட்டுவதற்கானவை. அதன் மூலம் அவர்கள் குழந்தைப் பருவ நலன் சார்ந்த செயல்பாடுகளுக்குத் திட்டமிட்டுச் செயல்படலாம்.
