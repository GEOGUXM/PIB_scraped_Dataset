      நிதி அமைச்சகம்

      போலி நிறுவனங்கள் மீது பணிக்குழு உயிர்ப்பான செயல்: தகவல்கள் திரட்டப்படுகின்றன
      Posted On: 08 JUN 2018 12:00PM by PIB Chennai


    போலி நிறுவனங்களின் மீது நடவடிக்கை எடுப்பதற்காக அமைக்கப்பட்ட பணிக்குழு ( Task Force ) தொடக்கம் முதல் இதுவரை 8 முறை கூடி விவாதித்து, உயிர்ப்பான சாதகமான பணியை நிறைவேற்றி வருகிறது. பிரதமர் அலுவலகத்தினால் 2017ம் ஆண்டு பிப்ரவரியில் அமைக்கப்பட்ட இந்தப் பணிக்குழு மத்திய வருவாய்த் துறைச் செயலர், கார்ப்பரேட் விவகார அமைச்சகச் செயலர் ஆகியோர் தலைமையில் இயங்கி வருகிறது.


    வரி ஏய்ப்பு செய்வதற்காகவும், சட்டவிரோதமாகச் செயல்படுவதற்காகவும் தொடங்கி நடத்தப்படும் போலி நிறுவனங்களை, பல்வேறு பிரிவுகளுடன் இணைந்து செயல்பட்டு முறையான வழிகளில் தடுத்து, நடவடிக்கை எடுப்பதற்காகவும் இந்தப் பணிக்குழு அமைக்கப்பட்டுள்ளது.


    இந்தக் குழுவில் நிதிச் சேவைகள் துறை, மத்திய நேரடி வரிகள் வாரியம் ( CBDT ), மத்திய கலால் மற்றும் சுங்கத் தீர்வை வாரியம் ( CBEC ), மத்திய புலனாய்வுக் கழகம் ( CBI ), அமலாக்க இயக்ககம் ( ED ), தீவிர மோசடிப் புலனாய்வு அலுவலகம் ( SFIO ), நிதிப் புலனாய்வுப் பிரிவு – இந்தியா ( FIU-IND ), இந்திய ரிசர்வ் வங்கி ( RBI ), இந்தியப் பங்கு வர்த்தக வாரியம் ( SEBI ) ஆகியவற்றின் அதிகாரிகளும், சரக்குகள்-சேவைகள் வரிப் புலனாய்வுத் துறை தலைமை இயக்குநர் ( DG GSTI), மத்திய பொருளாதாரப் புலனாய்வுப் பிரிவு தலைமை இயக்குநர் (DG-CEIB) ஆகியோரும் உறுப்பினர்கள் ஆவர்.


    போலி நிறுவனங்கள் குறித்த தகவல்கள் முழுவதையும் தீவிர மோசடிப் புலனாய்வு அலுவலகம் ( SFIO ) திரட்டியிருப்பது இந்தப் பணிக்குழுவின் மிகப் பெரிய சாதனையாகும். இன்றைய நிலவரப்படி இவ்வாறு திரட்டப்பட்ட தகவல்களில் மூன்று வகையான பட்டியல்கள் இடம்பெற்றுள்ளன. போலி என்று உறுதிப்படுத்தப்பட்ட நிறுவனங்கள், போலி என முடிவு செய்யப்பட்ட கம்பெனிகள், சந்தேகத்துக்கு உரியவை என்று மூன்று வகையான நிறுவனங்களின் பட்டியல்கள் தயாரிக்கப்பட்டுள்ளன. இவற்றில் பல்வேறு அமலாக்க நிறுவனங்களின் மூலம் பெறப்பட்ட தகவல்களின் அடிப்படையில் போலியானவை, சட்ட விரோதமான செயல்களில் ஈடுபட்டவை என்று 16,537 நிறுவனங்கள் உறுதிசெய்யப்பட்டுள்ளன. இரண்டாவது பட்டியலில் இடம்பெற்றுள்ள 16,739 போலி நிறுவனங்கள் ஒரே மாதிரியான இயக்குநர்களைக் கொண்டவையாக இருப்பது முழுமையாகக் கண்டறியப்பட்டு உறுதி செய்யப்பட்டுள்ளது. மூன்றாவதாக சந்தேகப் பட்டியலில் 80,670 நிறுவனங்கள் இடம்பெற்றுள்ளன. இவற்றைத் தீவிர மோசடிப் புலனாய்வு அலுவலகம் ( SFIO ) சில எச்சரிக்கை அறிகுறிகளை வைத்துக் கண்டறிந்துள்ளது. இதைப் போல் பணிக்குழுவின் அந்த அறிகுறிகளின் மூலம் மேலும் பல போலி நிறுவனங்கள் அடையாளம் கண்டறியப்படும்.


    மத்திய கார்ப்பரேட் விவகார அமைச்சகத்தின் மேற்பார்வையில் 2017-18ம் நிதியாண்டில் நடத்தப்பட்ட சோதனையில், கம்பெனிகள் சட்டம் (2013) 248ஆவது பிரிவின் கீழ் பதிவு செய்யப்பட்ட மற்றும் பதிவிலிருந்து நீக்கப்பட்ட கம்பெனிகளின் விவரங்கள் சேகரிக்கப்பட்டன. அவற்றில் 2,26,166 கம்பெனிகள் தங்களது நிதி அறிக்கைகளையோ ஆண்டு வருமான வரிக் கணக்குகளையோ தொடர்ந்து இரண்டு ஆண்டுகளுக்கு மேலாகத் தாக்கல் செய்யாதது கண்டறியப்பட்டது. அதைப் போல் கம்பெனிகள் சட்டம் (2013) 167(1) பிரிவு மற்றும் 164(2)(அ) பிரிவின் கீழ் 2013-14, 2014-15, 2015-16 ஆகிய ஆண்டுகளுக்கான நிதி அறிக்கைகளையோ வருமான வரிக் கணக்குகளையோ தாக்கல் செய்யாத 3,09,619 இயக்குநர்கள் தகுதியிழந்தவர்களாக அறிவிக்கப்பட்டனர். மத்திய கார்ப்பரேட் விவகாரங்கள் அமைச்சகம், நிதிச்சேவைகள் துறை, இந்திய வங்கிகள் சங்கம் ஆகியவற்றின் தொடர் முயற்சிகளால் அந்தக் கம்பெனிகளின் முன்னாள் இயக்குநர்களோ கையெழுத்திட வேண்டிய அதிகாரிகளோ குறிப்பிட்ட சில காரணங்கள் நீங்கலாக, வங்கிக் கணக்குகளைக் கையாள்வதற்குத் தடை விதிக்கப்பட்டுள்ளது. கம்பெனிகள் சட்டத்தின் 252வது பிரிவின் கீழ் விலக்கு அளிக்கப்படும் வரையில் இத்தடை நீடிக்கும். உண்மையான கார்ப்பரேட் நிறுவனங்கள் நிலுவையில் உள்ள தங்களது வருமான வரிக் கணக்குகளை முறைப்படுத்துவதற்கு உதவும் வகையில், 2018ம் ஆண்டு தாமதக் குற்றத்துக்கு மன்னிப்பு அளிக்கும் திட்டம் நடைமுறைக்கு வந்தது. இது 2018ம் ஆண்டு ஜனவரி 1ம் தேதி முதல் மே 1ம் தேதி வரையில் அமலில் இருந்தது. இத்திட்டத்தின் கீழ் 13,993 கம்பெனிகள் பலனடைந்தன.
