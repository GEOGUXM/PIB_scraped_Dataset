      நித்தி ஆயோக்

      இந்தியாவை மாற்றிய மகளிர் விருதுகள்: விண்ணப்பிக்க ஜூலை 31 கடைசி நாள்
      Posted On: 28 JUN 2018 3:54PM by PIB Chennai


    மூன்றாவது ஆண்டாக வழங்கப்படும் இந்தியாவை மாற்றிய பெண்களுக்கான விருதுக்கு ( WTI Awards ) விண்ணப்பிக்க வரும் ஜூலை 31ம் தேதி கடைசி நாள் என்று “நிதி ஆயோக்” அறிவித்துள்ளது. இந்த விருது ஐ.நா. சபை மற்றும் மும்பையில் உள்ள டைஸ் ( DICE ) அமைப்பு ஆகியவற்றுடன் இணைந்து அளிக்கப்படுகிறது. (டைஸ் ( DICE ) என்பது கல்வி, அறிவு, புதுமையாக்கம், தொழில்முனைவுக்கு உரிய சூழலை உருவாக்கும் அமைப்பாகும்).


    இந்தியாவை மாற்றிய மகளிர்க்கான விருது ( WTI Awards ) குறித்த அறிவிப்பு சர்வதேச மகளிர் தினத்தையொட்டி மார்ச் மாதம் வெளியிடப்பட்டது. தேசத்தைக் கட்டமைப்பதில் மிக உயர்வான சீரிய பங்களிப்பைச் செலுத்திய பெண்களுக்காக இந்த விருது வழங்கப்படுகிறது. இந்த ஆண்டு “மகளிரும் தொழில்முனைவுத் திறனும்” ( 'Women and Entrepreneurship' ) என்ற கருத்தில் இந்த விருது அளிக்கப்படுகிறது. நிதி ஆயோக் மகளிர் தொழில்முனைவுத் தளத்தை (WEP) தொடங்கியுள்ளது. அதையொட்டி இந்த விருது வழங்கப்படுகிறது.


    இந்த விருதுக்குத் தேர்ந்தெடுக்கப்படுபவருக்கு நிதி ஆயோக்கின் மகளிர் தொழில்முனைவுத் தளத்தின் கீழ் உதவியும் ஆதரவும் கிடைக்கும். இதன் மூலம் அவர்கள் தொழில் தொடங்கலாம்.


    வழக்கமான தொழில்கள் அல்லாமல் முற்றிலும் புதிய வணிகம், தொழிலில் ஈடுபடும் பெண்களுக்காக இந்த விருது வழங்கப்படுகிறது.

      * புதிய மேம்பாட்டுக்கான சவால்களுக்குப் புதிய தீர்வுகள் காணுதல் (அல்லது)
      * குறிப்பிட்ட பிரிவில் ஏற்படும் தாக்கம்

    இந்த விருதுகள் தொழில் வணிகத்தில் அடுத்து புதிதாக உருவாகும் புதுமைக்கு வழிகாட்டியாகி, சாதிக்கும் மகளிரின் திறமையை அங்கீகரிக்கும் வகையில் அளிக்கப்படுகின்றன. அந்த சாதனையின் மூலம் அப்பெண்கள் அதிகாரம் பெறுவதுடன் நாட்டுக்கும் வலிமை சேர்ப்பவராக அமைவர். நாடு முழுவதும் உள்ள மகளிர் தொழில்முனைவோர், பொருளாதார வளர்ச்சியில் பங்கேற்போர் இதற்கு விண்ணப்பிக்கலாம்.


    இதற்கான நிரப்பப்பட்ட படிவங்களைத் தனி நபர்களோ அவர்கள் சார்பில் மற்றவர்களோ கீழ்க்கண்ட இணையதளம் மூலம் விண்ணப்பிக்கலாம் :

      * மகளிர் தொழில்முனைவுத் தளம்: ( https://www.wep.gov.in/wti-awards ) அல்லது
      * அழைப்பு கொடுக்க வேண்டிய கைபேசி எண் : +91-704264887
