      நாடாளுமன்ற விவகாரத்துறை அமைச்சகம்

      நாடாளுமன்ற பட்ஜெட் கூட்டத்தொடர் நாளை தொடங்க உள்ள நிலையில் அனைத்துக் கட்சிக் கூட்டத்தை நடத்துகிறது அரசு
      தேசம் நல்லதை அடைய பட்ஜெட் கூட்டத்தொடரில் இணக்கமான சூழ்நிலையை உருவாக்க இணைந்து செயலாற்ற அனைத்து அரசியல் கட்சிகளுக்கும் திரு. நரேந்திர மோடி அழைப்பு
      இந்தியா முழுவதும் மாநில சட்டசபைகளுக்கு பங்கேற்பு ஜனநாயகத்திற்கான முன்மாதிரியாக இருக்கும்வகையில் நாடாளுமன்ற நிலைக்குழு முறையை நடைமுறைப்படுத்த பிரதமர் வலியுறுத்தல்
      68 நாட்களில் 31 அமர்வுகள் கொண்டதாக பட்ஜெட் கூட்டத்தொடர் இருக்கும்
      Posted On: 28 JAN 2018 6:55PM by PIB Chennai


    2018 ம் ஆண்டுக்கான நாடாளுமன்ற பட்ஜெட் கூட்டத் தொடர் நாளை தொடங்க உள்ள நிலையில் மக்களவை மற்றும் மாநிலங்களவை கட்சித் தலைவர்களிடையே இன்று உரையாற்றிய இந்தியப் பிரதமர் திரு . நரேந்திர மோடி , அனைத்து அரசியல் கட்சிகள் எழுப்பும் விவகாரங்களுக்கு அரசு பெரும் முக்கியத்துவம் அளிக்கிறது என்றும் , தேச நன்மைக்காக அனைத்து அரசியல் கட்சிகளும் பட்ஜெட் கூட்டத் தொடரில் இணக்கமான சூழ்நிலையை எட்ட கூட்டாக செயலாற்ற வேண்டும் என்றும் கேட்டுக் கொண்டார் .


    நாடாளுமன்றத்தின் பல்வேறு நிலைக் குழுக்கள் பெயரளவில் செயல்படுவதை சுட்டிக்காட்டிய திரு . மோடி இந்தக் குழுக்களின் முன் கொண்டுவரும் தேசிய முக்கியத்துவம் வாய்ந்த பிரச்சனைகளுக்கு உறுதியான தீர்வுகள் கிடைக்க ஆக்கப்பூர்வமான பங்களிப்பை அளி க் க வேண்டும் என அனைத்து அரசியல் கட்சிகளையும் ஊக்கப்படுத்தினார் . இந்தியாவில் உள்ள அனைத்து மாநில சட்டப்பேரவைகளிலும் பின்பற்றக் கூடிய பங்கேற்பு ஜனநாயககத்திற்கான முன்மாதிரியாக நாடாளுமன்ற குழுக்களை உருவாக்க கூடியிருந்த அனைத்து தலைவர்களையும் பிரதமர் கேட்டுக் கொண்டார் .


    இந்தக் கூட்டத்தில் பங்கேற்ற தலைவர்கள் ஏராளமான பிரச்சனைகளை எழுப்பினார்கள் . நாடாளுமன்றம் அமளி இன்றி சுமூகமாக செயல்படுவதை உறுதிப்படுத்தவும் இரு அவைகளிலும் ஆக்கப்பூர்வமான விவாதங்களின் மூலம் பிரச்சனைகளுக்கு தீர்வு காணவும் அனைத்து கட்சிகளிடையே ஒருமித்த கருத்து காணப்பட்டது .


    பின்னர் செய்தியாளர்களிடையே பேசிய மத்திய நாடாளுமன்ற விவகாரங்கள் துறை அமைச்சர் திரு . அனந்தகுமார் , அவை சுமூகமாக செயல்படுவதற்கு ஒத்துழைப்பு அளிக்குமாறு அனைத்து கட்சிகளையும் குறிப்பாக எதிர்க்கட்சியை அரசு கேட்டுக் கொண்டு இருப்பதாகத் தெரிவித்தார் . ஆக்கப்பூர்வமான பட்ஜெட் கூட்டத்தொடருக்கு அனைத்து கட்சிகளும் சாதகமாக இருப்பதாக அமைச்சர் கூறினார் . நடைமுறை விதிகள் அனுமதிக்கப்பட்டதன் அடிப்படையில் அவையில் விவாதம் நடத்த அரசு எப்போதும் தயாராக உள்ளது என அவர் மேலும் தெரிவித்தார் .


    நாடாளுமன்ற பட்ஜெட் கூட்டத்தொடர் 2018 ஜனவரி 29 ம் தேதி திங்கள்கிழமை தொடங்கும் என்றும் அரசுப் பணிகளின் தேவைக்கேற்ப 2018 ஏப்ரல் 6 வெள்ளிக்கிழமை நிறைவு பெறலாம் என்றும் திரு . அனந்தகுமார் கூறினார் . நாடாளுமன்ற நிலைக்குழுக்கள் பல்வேறு அமைச்சகங்கள் / துறைகளின் மானியக் கோரிக்கைகளை பரிசீலிப்பதற்காகவும் தங்களது அறிக்கைகளை அளிக்கவும் உதவும் வகையில் நாடாளுமன்றத்தின் இரு அவைகளும் 2018 பிப்ரவரி 9 ம் தேதி வெள்ளிக்கிழமை தற்காலிகமாக ஒத்திவைக்கப்பட்டு 2018 மார்ச் 5 ம் தேதி திங்கள்கிழமை மீண்டும் தொடங்கும்


    இந்தத் தொடரில் 31 அமர்வுகள் ( முதல் பகுதியில் 8 அமர்வுகள் மற்றும் இரண்டாவது பகுதியில் 23 அமர்வுகளும் ) இந்த 68 நாட்களில் நடக்கும் என அமைச்சர் குறிப்பிட்டார் .


    இந்த அமர்வு 2018 - 19 க்கான மத்திய பட்ஜெட் தொடர்பான நிதி விவகாரங்களில் முக்கிய கவனம் செலுத்துவதுடன் குடியரசுத் தலைவரின் உரைக்கு நன்றி தெரிவிக்கும் தீர்மானத்தின் மீதான விவாதம் குறித்தும் விவாதிக்கும் . எனினும் அவசியமான சட்ட விவகாரங்கள் மற்றும் இதர பிரச்சனைகள் குறித்தும் இந்த அமர்வில் விவாதிக்கப்படும் என்றும் திரு . அனந்த்குமார் தெரிவித்தார் .


    மேலும் ஜனவரி 29 ம் தேதி திங்கள் கிழமை நாடாளுமன்றத்தில் பொருளாதார ஆய்வறிக்கை தாக்கல் செய்யப்படும் என்றும் அமைச்சர் குறிப்பிட்டார் . 2018 - 19 மத்திய பட்ஜெட் 2018, பிப்ரவரி 1 ம் தேதி வியாழக்கிழமை மக்களவையில் தாக்கல் செய்யப்பட உள்ளது .


    இந்த அனைத்துக் கட்சிக் கூட்டத்தில் மத்திய உள்துறை அமைச்சர் திரு . ராஜ்நாத் சிங் , மத்திய நிதி அமைச்சர் திரு . அருண் ஜேட்லி , நாடாளுமன்ற விவகாரங்கள் மற்றும் புள்ளியியல் மற்றும் திட்ட அமலாக்கத்துறை இணையமைச்சர் திரு . விஜய் கோயல் , நாடாளுமன்ற விவகாரங்கள் மற்றும் நீர் ஆதாரங்கள் , நீர் மேம்பாடு மற்றும் கங்கை புதுப்பிப்புத் துறை இணையமைச்சர் திரு . அர்ஜுன் ராம் மேக்டால் மற்றும் அமைச்சர்கள் கலந்து கொண்டனர் .


    16 வது மக்களவையின் 13 வது அமர்வு மற்றும் மாநிலங்களவையில் 244 வது அமர்வின் முடிவில் மக்களவை மற்றும் மாநிலங்களவையில் நிலுவையில் உள்ள மசோதாக்கள் .

    2. தகவல் அளிப்போர் பாதுகாப்பு ( திருத்த ) மசோதா 2015


    3. நில கையகப்படுத்தலில் நியாயமான இழப்பீடு மற்றும் வெளிப்படைத்தன்மை , மறுவாழ்வு மற்றும் மீள் குடியேற்ற உரிமை ( திருத்தம் ) இரண்டாவது மசோதா 2015


    4. தொழிற்சாலைகள் ( திருத்த ) மசோதா 2015


    5. பிற்படுத்தப்பட்டோர் தேசிய ஆணையம் ( திரும்பப்பெறுதல் ) மசோதா 2017


    6. ஸ்டேட் வங்கி ( திரும்பப்பெறுதல் மற்றும் திருத்த ) மசோதா 2017


    7. பண்டைய கால நினைவிடங்கள் மற்றும் தொல்பொருள் இடங்கள் மற்றும் மீதங்கள் ( திருத்த ) மசோதா 2017


    8. மத்திய சாலைகள் நிதி ( திருத்த ) மசோதா 2017


    9. அசையாச் சொத்துக்கள் கையகப்படுத்துதலுக்கு கோரும் ( திருத்த ) மசோதா 2017


    10. இஸ்லாமிய பெண்கள் ( திருமணங்களில் உரிமை பாதுகாப்பு ) மசோதா 2017


    நிலைக்குழுவுக்கு பரிந்துரைக்கப்படாத மசோதாக்கள்


    11. தமிழ்நாடு சட்ட மேலவை ( திரும்பப்பெறும் ) மசோதா 2012


    12. நாடாளுமன்றம் மற்றும் சட்டமன்ற தொகுதிகளில் ஆதிதிரவிடர் மற்றும் பழங்குடியினர் பிரதிநிதித்துவத்தை மாற்றியமைத்தல் ( மூன்றாவது ) மசோதா 2013


    13. தில்லி வாடகை ( திரும்பபெறுதல் ) மசோதா 2013


    மக்களவையில் நிறைவேற்றப்பாட்டு தேர்வுக் குழுவுக்கு அனுப்பப்பட்ட மசோதா


    14. மோட்டார் வாகன ( திருத்த ) மசோதா 2017


    தேர்வுக் குழுவுக்கு பரிந்துரை செய்யப்பட்டு அறிக்கை அளிக்கப்பட்ட மசோதா


    15. ஊழல் தடுப்பு ( திருத்த ) மசோதா 2013


    நிலைக் குழுவால் அறிக்கை அளிக்கப்பட்ட மசோதாக்கள்


    16. அரசியல் சட்ட (79 வது திருத்த ) மசோதா 1992 ( உறுப்பினர்களுக்கான சிறு குடும்ப விதிகள் )


    17. தில்லி வாடகை ( திருத்த ) மசோதா 1997


    18. நகராட்சிகளுக்கான விதிகள் ( பட்டியலிடப்பட்ட பகுதிகளுக்கு விரிவாக்கம் ) மசோதா 2001


    19. விதை மசோதா 2004


    20. ஹோமியோபதி மத்திய கவுன்சில் ( திருத்த ) மசோதா 2005


    21. இந்திய மருத்துவ மையக் கவுன்சில் ( திருத்த ) மசோதா 2005


    22. இந்திய மருத்துவம் மற்றும் ஹோமியோபதி மருந்தக மசோதா 2005


    23. தனியார் புலனாய்வு முகமைகள் ( வரன்முறை ) மசோதா 2007


    24. பூச்சிக்கொல்லிகள் நிர்வாக மசோதா 2008


    25. இந்திய தொலைத்தொடர்பு வரன்முறை ஆணைய ( திருத்த ) மசோதா 2008


    26. சுரங்க திருத்த மசோதா 2011


    27. மாநிலங்களுக்கு இடையிலான தொழிலாளர் குடியேற்ற ( வேலைவாய்ப்பு மர்றும் சேவை நிபந்தனைகள் வரன்முறை ) திருத்த மசோதா 2011


    28. சுகாதாரத்திற்கான மனித வளத்திற்கான தேசிய ஆணைய மசோதா 2011


    29*. ஆயுதப் படைகள் நடுவர் மன்ற ( திருத்த ) மசோதா 2012


    30. பெண்களை தவறாக பிரதிநிதித்துவ ( தடுப்பு திருத்த ) மசோதா 2012


    31. கட்டிடம் மற்றும் இதர கட்டுமானத் தொழிலாளர்கள் தொடர்பான சட்டங்கள் ( திருத்த ) மசோதா 2013


    32. வேலை வாய்ப்பகங்கள் ( காலியிடங்கள் கட்டாய அறிவிப்பு ) மசோதா 2013


    33. ராஜஸ்தான் சட்டமேலவை மசோதா 2013


    34. இந்திய மருத்துவக் கவுன்சில் ( திருத்த ) மசோதா 2013


    35. நாளந்தா பல்கலைக்கழகத் ( திருத்த ) மசோதா 2013


    36. அசாம் சட்டமேலவை மசோதா 2013


    37. பதிவு ( திருத்த ) மசோதா 2013


    38. வக்ஃப் சொத்துக்கள் ( அங்கீகாரமற்ற ஆக்கிரமிப்பாளர்கள் அகற்றம் ) மசோதா 2014


    39. ஹோமியோபதி மத்திய சபைத் ( திருத்த ) மசோதா 2015
