      শেল-থুম মন্ত্রালয়

      সরকারনা এন.আই.এল. ফোর্ম জি.এস.তি.আর.-১ জুলাইগী অহানবা চয়োলদগী এস.এম.এস.কী খুত্থাংদা থাজিনবা য়াহল্লগনি
      Posted On: 27 JUN 2020 8:07PM by PIB Imphal


    জি.এস.তি. তেক্স থীরিবশিংদা খুদোংচানবগীদমক অচৌবা খোংথাং অমা ওইনা, সরকারনা ফোর্ম জি.এস.তি.-১গী এন.আই.এল. স্তেৎমেন্ত জুলাইগী অহানবা চয়োলদগী এস.এম.এস.কী খুত্থাংদা থাজিনবা য়াহনগদৌরি।


    সেন্ত্রেল বোর্দ ওব ইন্দাইরেক্ত তেক্স এন্দ কস্তম (সি.বি.আই.সি.)না হায়খ্রে মদুদি ফোর্ম জি.এস.তি.-১গী এন.আই.এল. স্তেৎমেন্তবু এস.এম.এস.কী খুত্থাংদা থাজিনবা য়াহনবনা লাখ ১২ হেন্না রেজিস্তর তৌরবা তেক্সপেয়রশিংনা জি.এস.তি.গী নিয়ম ইনবদা লাইহনলগনি। হৌজিক্কী তৌরিবসিদি তেক্সপেয়র খুদিংমক্না কম্মন পোর্তেলদা মখোয়গী একাউন্তদা লোগ ইন তৌগদবনি অমসুং মসিদগী ফোর্ম জি.এস.তি.-১দা আউৎৱার্দ সপ্লাইশিংগী স্তেৎমেন্ত অদু থা খুদিংগী নত্ত্রগা ক্বার্তর খুদিংগী ওইনা থাজিনগদবনি।


    এন.আই.এল. আউৎৱার্দ সপ্লাই লৈবা জি.এস.তি. তেক্স পেয়রশিংনা জি.এস.তি. পোর্তেলদা লোগ ওন তৌবগী তঙাইফদবা লৈররোই অমসুং মখোয়গী ফোর্ম জি.এস.তি.-১গী এন.আই.এল. স্তেৎমেন্ত অদু এস.এম.এস. অমগী খুত্থাংদা থাজিনবা ফাইল তৌবা য়ারগনি। ফাইল তৌখ্রবা স্তেৎমেন্ত অদুগী কনফার্মেসন স্তেতস নত্ত্রগা রিতর্ন এপ্লিকেসন অদু জি.এস.তি. পোর্তেলদা জি.এস.তি.আই.এন. একাউন্ত লোগ ইন তৌদুনা অমসুং সর্বিস>রিতর্ন>ত্রেক রিতর্ন স্তেতস্তা চঙদুনা ত্রেক তৌবা য়াগনি।


    খঙজিনবা য়াই মদুদি থাগী জি.এস.তি.আর.-৩বি. রিতর্নগী এন.আই.এল. ফাইল তৌবদি জুন ৮, ২০২০দগী হৌদুনা এস.এম.এস.কী খুত্থাংদা থাজিনবা য়াখ্রবনি। জি.এস.তি.আর.-৩বি. রিতর্নগী এন.আই.এল. ফাইল তৌগদবা লৈবা তেক্সপেয়রশিংনা খুদোংচাবা অসি লৌবা য়াগনি।


    এস.এম.এস.কী খুত্থাংদা থাজিন্ননবগীদমক, জি.এস.তি.আর.-৩বি. রিতর্নগী এন.আই.এল. ফাইল তৌগদবা তেক্সপেয়রশিংনা 14409 এস.এম.এস. অমা NIL<space>R1<space>GSTIN number<space>Tax period (MMYYYY দা) হায়না ইরগা থাজিনগদবনি। খুদম ওইনা NIL R1 09XXXXXXXXXXXZC 042020 (for monthly return of April 2020) নিত্ত্রগা NIL R1 09XXXXXXXXXXXZC 062020 (for quarterly return of April-June 2020)।


    মসিগী মতুংদা মখোয়দা মিনিৎ ৩০গী মতম খক্তগী ওইবা দিজিৎ তরুক্কী কোদ অমা লাক্কনি অমসুং মখোয়গী এন.আই.এল. স্তেতেন্ত ফাইলিং অদু 14409 দা CNF<space>R1<space> CODE নম্বীদুনা কনফার্ম তৌবা য়াগনি। কোদ অদু মায় পাক্না ভেলিদেৎ তৌরবা মতুংদা, রিতর্ন অদু ফাইল তৌরগনি অমসুং তেক্সপেয়র অদুদা এস.এম.এস.কী খুত্থাংদা একনোলেজমেন্ত নম্বর লাক্কনি।


    খোংথাং অসিগা মান্ননা তেক্সপেয়রশিংনা মখোয়গী জি.এস.তি.আর.-৩বি. রিতর্নগী এন.আই.এল. ফাইল অদু থাজিনবা য়াগনি, অদুমওইনমক 14409দা থাবা এস.এম.এস. অদুদা ৩বি.গী মহুত্তা আর.১ ওইনা ইগদবনি। অহেনবা অকুপ্পা মরোলগীদমক কনাগুম্বা অমনা www.gst.gov.in গী 'হেল্প' হায়বদুদা চঙদুনা খঙবা য়াগনি।
