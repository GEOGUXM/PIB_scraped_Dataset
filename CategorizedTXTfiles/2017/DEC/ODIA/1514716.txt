      ଔଦ୍ୟୋଗିକ ବ୍ୟାପାର ମନ୍ତ୍ରଣାଳୟ

      ଭାରତୀୟ ଦେବାଳିଆପଣ ବୋର୍ଡ ପ୍ରମୁଖ ସେବା ଲାଗି ବୈଷୟିକ ମାନକ ନିର୍ଦ୍ଦେଶାନାମା ଜାରି କଲେ
      Posted On: 14 DEC 2017 7:26PM by PIB Bhubaneshwar


    ଭାରତର ଦେବାଳିଆ ଏବଂ ଦେବାଳିଆପଣ ବୋର୍ଡ ( IBBI ) ପକ୍ଷରୁ ବୈଷୟିକ ସମିତି ସୁପାରିସ ଆଧାରରେ ଆଇବିବିଆଇ ( ସୂଚନା ଉପଯୋଗିତା ) ନିୟମ , 2017 ର ଧାର୍ଯ୍ୟନିୟମ 13 ଅନ୍ତର୍ଗତ ନିମ୍ନଲିଖିତ ମାମଲାଗୁଡ଼ିକ ପାଇଁ ପ୍ରମୁଖ ସେବାଗୁଡ଼ିକର ପ୍ରଦର୍ଶନ ନିମନ୍ତେ ବୈଷୟିକ ମାନକ ନିର୍ଦ୍ଧାରଣ କରାଯାଇଛି :


    i ) ସେବାର ମାନକ ସର୍ତ୍ତ ;


    ii ) ଉପଯୋଗକର୍ତ୍ତାଙ୍କ ପଞ୍ଜିକରଣ ;


    iii ) ପ୍ରତ୍ୟେକ ରେକର୍ଡ ଏବଂ ପ୍ରତ୍ୟେକ ଉପଯୋଗକର୍ତ୍ତା ନିମନ୍ତେ ବିଶିଷ୍ଟ ପରିଚୟକର୍ତ୍ତା ;


    iv ) ସୂଚନା ଦାଖଲ କରିବା ;


    v ) ବ୍ୟକ୍ତିଙ୍କ ଚିହ୍ନଟ ଏବଂ ଯାଞ୍ଚ ;


    vi ) ସୂଚନା ପ୍ରମାଣିତ କରିବା ;


    vii ) ସୂଚନାର ଯାଞ୍ଚ ;


    viii ) ଡାଟାର ଶୁଦ୍ଧତା


    ix ) ତୃତୀୟ କିମ୍ବା ଅନ୍ୟ ପକ୍ଷଙ୍କୁ ସୂଚନା ପର୍ଯ୍ୟନ୍ତ ପହଁଚ ପ୍ରଦାନ କରିବା ନିମନ୍ତେ ସହମତିର ରୂପରେଖ ;


    x ) ପ୍ରଣାଳୀର ସୁରକ୍ଷା ;


    xi ) ସୂଚନାର ସୁରକ୍ଷା ;


    xii ) ବିପତି ପରିଚାଳନାର ରୂପରେଖ ;


    xiii ) ସୂଚନାର ସଂରକ୍ଷଣ ; ଏବଂ


    xiv ) ସୂଚନାର ଶୁଦ୍ଧୀକରଣ ।


    ସେବା ପ୍ରଦାନ କରିବା ସମୟରେ କୌଣସି ସୂଚନା ଉପଯୋଗିତା କିମ୍ବା ଉପକ୍ରମ ଲାଗୁ ବୈଷୟିକ ମାନକର ପାଳନ କରିବା ।


    ନିର୍ଦ୍ଦେଶନାମାଗୁଡ଼ିକ www.mca.gov.in and www.ibbi.gov.in . ରେ ଉପଲବ୍ଧ ।
