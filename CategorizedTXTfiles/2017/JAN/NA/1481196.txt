      মন্ত্রীমন্দল মীফমগী ৱারেপ

      কেন্দ্রগী মন্ত্রীমন্দলনা চৎনদ্রবা অমসুং মথৌ তাদবা কাংলোনশিং ককথৎনবা অয়াবা পীখ্রে
      Posted On: 18 JAN 2017 3:23PM by


    


    প্রধানমন্ত্রী শ্রী নরেন্দ্র মোদীনা লুচীংবা কেন্দ্রগী মন্ত্রীমন্দলনা এক্ত ১০৫ ককথৎনবগীদমক রিপীলিং অমসুং এমেন্দমেন্ত বিল, ২০১৭ হৌদোক্নবা মখোয়গী অয়াবা পীখ্রে ।

    য়ুমফমঃ

    প্রধানমন্ত্রী ওফিস ( পি এম ও )না শেম্বা মীহুৎ অনি লৈবা ভারতকী লো কমিসন অমসুং লেজিসলেতিব বিভাগনা ককথৎনবগীদমক চৎনদ্রবা অমসুং মথৌ তাদবা সেন্ত্রেল এক্ত ১৮২৪ খঙদোক্লে । ভারতক সরকারগী তোঙান তোঙানবা মন্ত্রালয়/বিভাগশিংগা লোইননা খন্ন-নৈনরবা অমসুং থিজিল্লবা মতুংদা সেন্ত্রেল এক্ত ১১৭৫ ( মে ২০১৪ দগী ওগস্ত ২০১৬ ফাওবা ) ককথৎনবগীদমক পার্লিয়ামেন্তনা এক্ত মরি চৎনহনখ্রে । মখোয়শিংদুদি -

    i. রিপীলিং অমসুং এমেন্দিং এক্ত, ২০১৫ ( ২০১৫গী ১৭শুবা ) এক্ত ৩৫ ককথৎপা, রিপীলিং অমসুং এমেন্দিং এক্ত ( ২শুবা ) এক্ত, ২০১৫ ( ২০১৫গী ১৯শুবা ) এক্ত ৯০ ককথৎপা,

    ii. এপ্রোপিয়েসন এক্ত ( রিপীল ) এক্ত, ২০১৬ ( ২০১৬কী ২২শুবা ) এক্ত ৭৫৬ ককথৎপা,

    iii. এপ্রোপিয়েসন ( রেলৱেজ ) এক্ত য়াওনা এপ্রোপিয়েসন এক্ত,
    iv. রিপীলিং অমসুং এমেন্দিং এক্ত, ২০১৬ ( ২০১৬কী ২৩শুবা ) এক্ত ২৯৪ ককথৎপা ।
