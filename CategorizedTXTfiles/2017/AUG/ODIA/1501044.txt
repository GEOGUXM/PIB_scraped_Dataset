      ଯୁବ ବ୍ୟାପାର ଓ କ୍ରୀଡା ମନ୍ତ୍ରଣାଳୟ

      ଜାତୀୟ କ୍ରୀଡ଼ା ଏବଂ ଦୁଃସାହାସିକ ପୁରସ୍କାର-2017
      Posted On: 29 AUG 2017 8:49PM by PIB Bhubaneshwar


    ଭାରତର ରାଷ୍ଟ୍ରପତି ଶ୍ରୀ ରାମନାଥ କୋବିନ୍ଦ ଆଜି ରାଷ୍ଟ୍ରପତି ଭବନରେ ଜାତୀୟ କ୍ରୀଡ଼ା ଏବଂ ଦୁଃସାହାସିକ ପୁରସ୍କାର 2017 ପ୍ରଦାନ କରିଛନ୍ତି । ଏହି ଅବସରରେ ଯୁବବ୍ୟାପାର ଓ କ୍ରୀଡ଼ା ( ସ୍ୱାଧୀନ ) ରାଷ୍ଟ୍ରମନ୍ତ୍ରୀ ଶ୍ରୀ ବିଜୟ ଗୋଏଲ ଉପସ୍ଥିତ ଥିଲେ ।


    ପୁରସ୍କାର ଉତ୍ସବରେ ଯୋଗଦେଇଥିବା 14 ଜଣ ସଫଳ କ୍ରୀଡ଼ାବିତଙ୍କ ସହିତ ରାଷ୍ଟ୍ରପତି ଶ୍ରୀ କୋବିନ୍ଦ ମଧ୍ୟ ଭାବ ବିନିମୟ କରିଥିଲେ । ସେମାନଙ୍କ ମଧ୍ୟରେ ଥିଲେ ଏସିଆନ ସ୍କୁଲ ଚାମ୍ପିଅନସିପ ଫର ଆଥଲେଟିକ୍ସ 2015ରେ ପଦକ ଜିତିଥିବା 4 ଜଣ ସ୍ୱର୍ଣ୍ଣପଦକ ବିଜୟୀ କ୍ରୀଡାବିତ , ଏସିଆନ ସ୍କୁଲ ଚାମ୍ପିଅନସିପ ଫର ସୁଇମିଂ 2016ରେ ସ୍ୱର୍ଣ୍ଣପଦକ ଜିତିଥିବା ଜଣେ ସନ୍ତରଣକାରୀ ଏବଂ 2016ରେ ଇଣ୍ଟରନେସନାଲ ସ୍କୁଲ ସ୍ପୋର୍ଟ୍ସ ଫେଡେରେସନ ଦ୍ୱାରା ଆୟୋଜିତ ଜିମ୍ନାସିଆଡରେ 9 ଜଣ ସ୍ୱର୍ଣ୍ଣପଦକ ବିଜୟୀ ଖେଳାଳି ।


    କ ) ରାଜୀବ ଗାନ୍ଧୀ ଖେଳରତ୍ନ 2017


    ନିମ୍ନଲିଖିତ ଖେଳାଳିମାନଙ୍କୁ ରାଜୀବ ଗାନ୍ଧୀ ଖେଳରତ୍ନ 2017 ପୁରସ୍କାର ପ୍ରଦାନ କରାଯାଇଛି ।


    1 ) ସର୍ଦ୍ଦାରା ସିଂ ( ହକି )


    2 ) ଦେବେନ୍ଦ୍ର ( ଦିବ୍ୟାଙ୍ଗ କ୍ରୀଡ଼ାବିତ )


    ଖ ) ଦ୍ରୋଣାଚାର୍ଯ୍ୟ ପୁରସ୍କାର 2017


    ନିମ୍ନଲିଖିତ କୋଚ୍ ମାନଙ୍କୁ ଦ୍ରୋଣାଚାର୍ଯ୍ୟ ପୁରସ୍କାର 2017 ପ୍ରଦାନ କରାଯାଇଛି ।


    1 ) ସ୍ୱର୍ଗତ ଡଃ ଆର ଗାନ୍ଧୀ ( ଦୌଡକୁଦ )


    2 ) ଜିଏସଏସଭି ପ୍ରସାଦ ( ବେଡମିଣ୍ଟନ ) ଲାଇଫଟାଇମ


    3 ) ବ୍ରଜ ଭୁଷଣ ମହାନ୍ତି ( ବକ୍ସିଂ ) ଲାଇଫଟାଇମ


    4 ) ପିଏ ରାଫେଏଲ ( ହକି ) ଲାଇଫଟାଇମ


    5 ) ସଞ୍ଜୟ ଚକ୍ରବର୍ତୀ ( ବନ୍ଧୁକ ଚାଳନା ) ଲାଇଫଟାଇମ


    6 ) ରୋଶନ ଲାଲ ( କୁସ୍ତି ) ଲାଇଫଟାଇମ


    ସ୍ୱର୍ଗତ ଡଃ ଆର ଗାନ୍ଧୀଙ୍କ ପୁରସ୍କାର ତାଙ୍କ ପତ୍ନୀ ଡଃ ସାଇଲକ୍ଷ୍ମୀ ଗାନ୍ଧୀ ଗ୍ରହଣ କରିଥିଲେ ।


    ଗ ) ଅର୍ଜୁନ ପୁରସ୍କାର


    ନିମ୍ନଲିଖିତ କ୍ରୀଡ଼ାବିତଙ୍କୁ 2017 ବର୍ଷ ପାଇଁ ଅର୍ଜୁନ ପୁରସ୍କାର ପ୍ରଦାନ କରାଯାଇଛି ।


    1 ) ଭେନମ ଜ୍ୟୋତି ସୁରେଖା ( ଧନୁର୍ବିଦ୍ୟା )


    2 ) ଖୁସବୀର କୌର ( ଦୌଡକୁଦ )


    3 ) ଆରୋକିଆ ରାଜୀବ ( ଦୌଡକୁଦ )


    4 ) ସୁଶ୍ରୀ ପ୍ରଶାନ୍ତି ସିଂ ( ବାସ୍କେଟବଲ )


    5 ) ଲାଇସାରାମ ଦେବେନେ୍ଦ୍ରା ସିଂ ( ବକ୍ସିଂ )


    6 ) ଚେତେଶ୍ୱର ପୁଜାରା ( କ୍ରିକେଟ )


    7 ) ହରମନପ୍ରୀତ କୌର ( ମହିଳା କ୍ରିକେଟ )


    8 ) ଓଇନାମ ବେମବେମ ଦେବୀ ( ମହିଳା ଫୁଟବଲ )


    9 ) ଶିବଶଙ୍କର ପ୍ରସାଦ ଚୌରାସିଆ ( ଗଲ୍ଫ )


    10 ) ଏସ ଭି ସୁନିଲ ( ହକି )


    11 ) ଯଶବୀର ସିଂ ( କବାଡି )


    12 ) ପିଏନ ପ୍ରକାଶ ( ସୁଟିଂ )


    13 ) ଏ ଅମଲରାଜ ( ଟେବୁଲ ଟେନିସ )


    14 ) ସାକେଥ ସାଇ ମିନେନି ( ଟେନିସ )


    15 ) ସତ୍ୟବ୍ରତ କାଡିଆନ ( କୁସ୍ତି )


    16 ) ମାରିଆପ୍ପାନ ଟି ( ଦିବ୍ୟାଙ୍ଗ କ୍ରୀଡାବିତ )


    17 ) ବରୁଣ ସିଂ ଭଟି ( ଦିବ୍ୟାଙ୍ଗ କ୍ରୀଡାବିତ )


    କ୍ରିକେଟ କ୍ଷେତ୍ରରେ ଅର୍ଜୁନ ପୁରସ୍କାର ପାଇଁ ଚୟନ ହୋଇଥିବା ଚେତେଶ୍ୱର ପୁଜାରା ଏବେ ବିଦେଶରେ ଖେଳୁଥିବାରୁ ପୁରସ୍କାର ଉତ୍ସବରେ ଯୋଗଦେଇ ପାରିନଥିଲେ । ପରବର୍ତ୍ତୀ ସମୟରେ ତାଙ୍କୁ ଯୁବସେବା ଓ କ୍ରୀଡ଼ା ରାଷ୍ଟ୍ରମନ୍ତ୍ରୀ ( ସ୍ୱାଧୀନ ) ଙ୍କ ଦ୍ୱାରା ଏହି ପୁରସ୍କାର ପ୍ରଦାନ କରାଯିବ ।


    ଘ ) ଧ୍ୟାନଚାନ୍ଦ ପୁରସ୍କାର


    ନିମ୍ନଲିଖିତ ଏହି ବ୍ୟକ୍ତି ମାନଙ୍କୁ 2017 ବର୍ଷ ପାଇଁ ଧ୍ୟାନଚାନ୍ଦ ପୁରସ୍କାର ପ୍ରଦାନ କରାଯାଇଛି ।


    1 ) ଭୁପିନ୍ଦର ସିଂ ( ଦୌଡକୁଦ )


    2 ) ସୟଦ ସହିଦ ହକିମ ( ଫୁଟବଲ )


    3 ) ସୁମାରି ଟେଟେ ( ହକି )


    ଙ ) ତେନଜିଂ ନର୍ଗେ ଜାତୀୟ ଦୁଃସାହାସିକ ପୁରସ୍କାର


    ନିମ୍ନଲିଖିତ ବ୍ୟକ୍ତିମାନଙ୍କୁ 2017 ବର୍ଷ ପାଇଁ ତେନଜିଂ ନର୍ଗେ ଜାତୀୟ ଦୁଃସାହାସିକ ପୁରସ୍କାର ପ୍ରଦାନ କରାଯାଇଛି ।


    1 ) ପ୍ରେମଲତା ଅଗ୍ରୱାଲ ( ସ୍ଥଳଭାଗ ଦୁଃସାହାସିକତା )


    2 ) ମୋରେ ରୋହନ ଦତାତ୍ରେୟ ( ଜଳଭାଗ ଦୁଃସାହାସିକତା )


    3 ) ଅଶୋକ ଆବେ ( ଲାଇଫ ଟାଇମ ଏଚିଭମେଣ୍ଟ )


    4 ) ବେଦ ପ୍ରକାଶ ଶର୍ମା ( ଲାଇଫ ଟାଇମ ଏଚିଭମେଣ୍ଟ )


    ଚ ) ମୌଲାନା ଅବୁଲ କଲାମ ଆଜାଦ ( ଏମଏକେଏ ) ଟ୍ରଫି 2016-17


    ପଞ୍ଜାବି ବିଶ୍ୱବିଦ୍ୟାଳୟ , ପଟିଆଲାକୁ 2016-17ବର୍ଷ ପାଇଁ ମୌଲାନା ଅବଦୁଲ କଲାମ ଆଜାଦ ଟ୍ରଫି ପ୍ରଦାନ କରାଯାଇଛି ।


    1 ) ପଞ୍ଜାବି ବିଶ୍ୱବିଦ୍ୟାଳୟ , ପଟିଆଲା ( ବିଜୟୀ )


    ଛ ) ରାଷ୍ଟ୍ରୀୟ ଖେଳ ପ୍ରୋତ୍ସାହନ ପୁରସ୍କାର , 2017


    ନିମ୍ନଲିଖିତ ଅନୁଷ୍ଠାନମାନଙ୍କୁ ଖେଳ ପ୍ରୋତ୍ସାହନ ପୁରସ୍କାର 2017 ପ୍ରଦାନ କରାଯାଇଛି ।


    1 ) କେନ୍ଦ୍ରୀୟ ବିଦ୍ୟାଳୟ ସଙ୍ଗଠନ ( କେଭିଏସ ) ( ନବୀନ ଓ ଯୁବ ପ୍ରତିଭାଙ୍କ ଚିହ୍ନଟ ଓ ପ୍ରଶିକ୍ଷଣ )


    2 ) ଓଡିଶା ଶିଳ୍ପ ଉନ୍ନୟନ କର୍ପୋରେସନ ( ଇଡକୋ ) କର୍ପୋରେଟ ସାମାଜିକ ଦାୟିତ୍ୱବୋଧ ମାଧ୍ୟମରେ କ୍ରୀଡ଼ାକୁ ପ୍ରୋତ୍ସାହନ


    3 ) ଦ ଗଲ୍ଫ ଫାଉଣ୍ଡେସନ ( ଉନ୍ନୟନ ଲାଗି କ୍ରୀଡ଼ା )


    4 ) ରିଲାଏନ୍ସ ଫାଉଣ୍ଡେସନ ( ଉନ୍ନୟନ ଲାଗି କ୍ରୀଡ଼ା )
