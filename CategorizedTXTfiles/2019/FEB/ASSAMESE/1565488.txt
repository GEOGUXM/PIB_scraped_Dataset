      কেবিনেট

      ৰে’ল বিভাগৰ গ্ৰুপ ‘এ’ পৰ্যায়ৰ ছাৰ্ভিছৰ ৮টা সংগঠিত গোটৰ কেডাৰ পৰ্যালোচনাত কেবিনেটৰ অনুমোদন
      Posted On: 19 FEB 2019 8:52PM by PIB Guwahati


    প্ৰধানমন্ত্রী শ্ৰী নৰেন্দ্ৰ মোদীৰ অধ্যক্ষতাত অনুষ্ঠিত কেন্দ্রীয় কেবিনেট বৈঠকত ইণ্ডিয়ান ৰে ’ লৱে একাউণ্টছ ছাৰ্ভিচ ( আইআৰএএছ ), ইণ্ডিয়ান ৰে ’ লৱে পাৰ্ছনেল ছাৰ্ভিচ ( আইআৰএএছ ), ইণ্ডিয়ান ৰে ’ লৱে ট্ৰেফিক ছাৰ্ভিচ ( আইআৰটিএছ ), ইণ্ডিয়ান ৰে ’ লৱে ছাৰ্ভিচ অফ ইঞ্জিনিয়াৰ্ছ ( আইআৰএছই ), ইণ্ডিয়ান ৰে ’ লৱে ছাৰ্ভিচ অফ ইলেক্ট্রিকেল ইঞ্জিনিয়াৰ্ছ ( আইআৰএছইই ), ইণ্ডিয়ান ৰে ’ লৱে ছাৰ্ভিচ অফ মেকানিকেল ইঞ্জিনিয়াৰ্ছ ( আইআৰএছএমই ), ইণ্ডিয়ান ৰে ’ লৱে ষ্টোৰ্ছ ছাৰ্ভিচ ( আইআৰএছএছ ), ইণ্ডিয়ান ৰে ’ লৱে ছাৰ্ভিচ অফ ছিগনেল ইঞ্জিনিয়াৰ্ছ ( আইআৰএছএছই ) ৰে ধৰি ৮টা সংগঠিত গ্রুপ ‘ এ ’ পৰ্যায়ৰ ছাৰ্ভিচ কেডাৰ পর্যালোচনাৰ প্ৰস্তাৱ অনুমোদন জনাইছে৷


    তদুপৰি , আইআৰপিএছ কেডাৰৰ এটি পদ ( সদস্যকর্মী ), মহানির্দেশক ( ছিগনেল আৰু টেলিকম ), মহানির্দেশক ( ষ্টোৰ্ছ ), মহানির্দেশক ( নিৰাপত্তা ) পদসমূহৰ নাম সলনি কৰাৰ সিদ্ধান্ততো কেবিনেটে অনুমোদন জনাইছে।


    ২০১২চনৰ পৰা কেডাৰ পর্যালোচনাৰ কাম বাদ পৰি আছিল আৰু এই পর্যালোচনাৰ ফলত সংশ্লিষ্ট ছাৰ্ভিচসমূহৰ ৯০০গৰাকী বিষয়া উপকৃত হ ’ ব৷
