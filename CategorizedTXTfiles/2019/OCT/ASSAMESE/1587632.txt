      কেবিনেট

      কেবিনেটক এন এইচ এমৰ অধীনত প্ৰগতি তথা এন এইচ এমৰ ই পি চি তথা এম এছ জিৰ নিৰ্ণয়সমূহৰ বিষয়ে অৱগত কৰোৱা হৈছে
      Posted On: 09 OCT 2019 2:44PM by PIB Guwahati


    প্ৰধানমন্ত্ৰী শ্ৰী নৰেন্দ্ৰ মোদীৰ পৌৰহিত্যত কেন্দ্ৰীয় কেবিনেটক ৰাষ্ট্ৰীয় স্বাস্থ্য অভিযান ( এন এইচ এম ) ৰ অধীনত প্ৰগতি তথা এন এইচ এমৰ অধিকৃত কাৰ্যক্ৰম সমিতি ( ই পি চি ) তথা অভিযান সঞ্চালন গোট ( এম এছ জি ) ৰ নিৰ্ণয়সমূহৰ বিষয়ে অৱগত কৰোৱা হয়৷


    মুখ্য বিশেষত্বসমূহঃ

      * এন আৰ এইচ এম / এন এইচ এমৰ সূচনাৰ পিছত মাতৃ মৃত্যু দৰ ( এম এম আৰ ), আণ্ডাৰ ফাইভ মৰ্টেলিটী ৰেট ( ইউ ৫ এম আৰ ) আৰু আই এম আৰত ব্যাপক হ্ৰাস পৰিলক্ষিত হৈছে৷
      * মেলেৰিয়া অধ্যূষিত দেশসমূহৰ ভিতৰতে ভাৰতে এই প্ৰত্যাহ্বান মোকাবিলা কৰাত যথেষ্ট সফলতা লাভ কৰিছে৷
      * সংশোধিত ৰাষ্ট্ৰীয় যক্ষ্মাৰোগ নিয়ন্ত্ৰণ কাৰ্যসূচী ( আৰ এন টি চি পি ) ৰ সৱলীকৰণ কৰা হৈছে৷ সকলো জিলাত মুঠ ১১৮০ চি বি এন এ এ টি মেচিন লগোৱা হৈছে৷
      * ২০১৮ - ১৯ত ৫২৭৪৪ এ বি - এইচ ডব্লিউ চিৰ অনুমোদন জনোৱা হৈছে যাৰ অধীনত ১৫০০০ ৰ লক্ষ্যৰ প্ৰতি ১৭১৪৯ এইচ ডব্লিউ চিৰ সঞ্চালন কৰা হৈছে৷
      * নতুন টীকাসমূহৰ মাজত টিটেনাচ আৰু বয়স্ক ডিপথেৰিয়া ভেকচিনে টিটেনাচ টক্সাইড ভেকচিনৰ স্থান গ্ৰহণ কৰিছে৷
      * ২০১৮ত ১৭খন অতিৰিক্ত ৰাজ্যত মীজেল্স - ৰুবেলা ( এম আৰ ) টীকাকৰণৰ অভিযান চলোৱা হৈছিল৷
      * ২০১৮ - ১৯ৰ সময়চোৱাত ৰোটা ভাইৰাছ ভেকচিন অতিৰিক্ত দুখন ৰাজ্যত আৰম্ভ কৰা হৈছিল৷ আজি ইয়াত সকলো ৰাজ্য / কেন্দ্ৰীয় শাসিত অঞ্চল অন্তৰ্ভুক্ত হৈছে৷
      * পোষণ অভিযানৰ অধীনত এপ্ৰিল ২০১৮ত এনিমিয়ামুক্ত ভাৰত ( এ এম বি ) অভিযান আৰম্ভ কৰা হৈছিল৷
      * এইচ ডব্লিউ চিত উপ স্বাস্থ্যকেন্দ্ৰসমূহৰ বাবে একীকৃত ধনৰাশি ২০০০০ টকাৰ পৰা বৃদ্ধি কৰি ৫০০০০ টকা কৰা হৈছে৷
      * টিবি / কুষ্ঠ ৰোগ / মেলেৰিয়া / ক ’ লা জ্বৰ আদি ৰোগৰ নিৰ্মূলৰ বাবে ৰাজ্য / সংঘ ৰাজ্য ক্ষেত্ৰ / জিলাসমূহক পুৰষ্কৃত কৰাৰ কথা ঘোষণা কৰা হৈছে৷
