      کامرس اور صنعت کی وزارتہ

      آٹھ کلیدی صنعتوں کا عدد اشاریہ (بنیاد : 2011-2012=100) نومبر 2019
      Posted On: 31 DEC 2019 5:03PM by PIB Delhi


    نئیدہلی یکم جنوری 2020۔آٹھ کلیدی صنعتوں کے عدداشاریہ کا مختصر خاکہ ( بنیاد : دوہزار گیارہ بارہ ) ضمیمے میں دیا گیا ہے ۔


    2۔ آٹھ کلیدی صنعتیں ، جو صنعتی پیداوار کے اشاریہ میں شامل کی گئی اشیا میں آٹھ بنیادی صنعتوں کا حصہ 40.27 کے بقدر ہے۔ آٹھ بنیادی صنعتوں کا تشکیل شدہ اشاریہ ماہ نومبر 2019 میں 126.3 تھا جو نومبر2018 کے مقابلے گھٹ کر 1.5 کے بقدر ہوگیاہے۔ اپریل تا نومبر 20-2019 کے دوران اس کی مجموعی شرح نمو 0.0 فیصد تھی۔


    کوئلہ :


    کوئلے کی پیداوار ( حصہ : 10.33 فیصد ) میں نومبر 2018 کی بہ نسبت نومبر 2019 میں 2.5فیصد تک کمی آئی ہے ۔ اس کی مجموعی پیداوار کےا نڈیکس میں اپریل سے نومبر 20-2019 کی مدت میں پچھلی سال کی اسی مدت کی بہ نسبت 5.3فیصد کمی آئی ہے ۔


    خام تیل :


    خام تیل کی پیداوار میں ( حصہ : 8.98 فیصد ) میں نومبر 2018 کی بہ نسبت نومبر 2019 میں 6.0فیصد) کی کمی آئی ہے ۔ اس کی مجموعی پیداوار کےا نڈیکس میں اپریل سے نومبر 20-2019 کی مدت میں پچھلی سال کی اسی مدت کی بہ نسبت 5.9فیصد کمی آئی ہے ۔


    قدرتی گیس :


    قدرتی گیس کی پیداوار میں ( حصہ : 6.88 فیصد ) میں نومبر 2018 کی بہ نسبت نومبر 2019 میں 6.4فیصد) کی کمی آئی ہے ۔ اس کی مجموعی پیداوار کےا نڈیکس میں اپریل سے نومبر 20-2019 کی مدت میں پچھلی سال کی اسی مدت کی بہ نسبت 3.1فیصد کمی آئی ہے ۔


    ریفائنری پیداوار :


    ریفائنری پیداوار میں ( حصہ : 28.04 فیصد ) میں نومبر 2018 کی بہ نسبت نومبر 2019 میں 3.1فیصد) کی کمی آئی ہے ۔ اس کی مجموعی پیداوار کےا نڈیکس میں اپریل سے نومبر 20-2019 کی مدت میں پچھلی سال کی اسی مدت کی بہ نسبت 1.1فیصد کمی آئی ہے ۔


    کیمیاوی کھادیں :


    کیمیاوی کھادوں کی پیداوار میں ( حصہ : 2.63 فیصد ) میں نومبر 2018 کی بہ نسبت نومبر 2019 میں 13.6فیصد) کی کمی آئی ہے ۔ اس کی مجموعی پیداوار کےا نڈیکس میں اپریل سے نومبر 20-2019 کی مدت میں پچھلی سال کی اسی مدت کی بہ نسبت 4.0فیصد کمی آئی ہے ۔


    اسٹیل :


    اسٹیل کی پیداوار میں ( حصہ : 17.92 فیصد ) میں نومبر 2018 کی بہ نسبت نومبر 2019 میں 3.7فیصد) کی کمی آئی ہے ۔ اس کی مجموعی پیداوار کےا نڈیکس میں اپریل سے نومبر 20-2019 کی مدت میں پچھلی سال کی اسی مدت کی بہ نسبت 5.2فیصد کمی آئی ہے ۔


    سیمنٹ :


    سیمنٹ کی پیداوار میں ( حصہ : 5.37 فیصد ) میں نومبر 2018 کی بہ نسبت نومبر 2019 میں 4.1فیصد) کی کمی آئی ہے ۔ اس کی مجموعی پیداوار کےا نڈیکس میں اپریل سے نومبر 20-2019 کی مدت میں پچھلی سال کی اسی مدت کی بہ نسبت 0.02فیصد کمی آئی ہے ۔


    بجلی :


    بجلی کی پیداوار میں ( حصہ : 19.85فیصد ) میں نومبر 2018 کی بہ نسبت نومبر 2019 میں 5.7فیصد) کی کمی آئی ہے ۔ اس کی مجموعی پیداوار کےا نڈیکس میں اپریل سے نومبر 20-2019 کی مدت میں پچھلی سال کی اسی مدت کی بہ نسبت 0.7فیصد کمی آئی ہے ۔


    نوٹ :

     1. ستمبر 2019 ، اکتوبر 2019 اور نومبر 2019 کا ڈیٹا عارضی ہے ۔
     2. اپریل 2014سے قابل تجدید وسائل کے بجلی کی پیداور کے ڈیٹا کو بھی شامل کیا گیا ہے ۔
     3. مذکورہ بالا صنعت کے لحاظ سے ظاہر کئے گئے حصص یا اوزان اپنی نوعیت کے لحاظ سے ہرصنعت کے ساتھ انفرادی حصص والے ہیں اور آئی آئی پی سے حاصل کئے گئے اور مساوی 100 کے اصول کے مقابق مقدار کی مناسبت سے یکجا کرکے پیش کیا گیا ہے ۔
     4. مارچ 2019 سے جن کوائلز کو ٹھنڈا کرکے مدوّر بنایا گیا ہے ، اُن اشیا کے تحت پاٹ رولڈ ٹکلڈ اور آئلڈ ( ایچ آر پی او ) کہلایا جانے والی ایک نئی اسٹیل کی پیداوار کو بھی شامل کیا گیا ہے ۔
     5. دسمبر 2019 کے عدد اشاریہ کا اجرا جمعہ 31 جنوری 2020 کو ہوگا۔
