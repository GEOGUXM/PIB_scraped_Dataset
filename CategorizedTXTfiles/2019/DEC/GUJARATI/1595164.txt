      મહિલા અને બાળ વિકાસ મંત્રાલય

      ગુજરાતમાં હાલ કાર્યરત ‘વન સ્ટોપ સેન્ટર’ની જિલ્લાવાર સ્થિતિ
      Posted On: 05 DEC 2019 6:29PM by PIB Ahmedabad


    રાજ્યસભામાં પ્રશ્નકાળ દરમિયાન આજે (5 ડિસેમ્બર, 2019) સાંસદ શ્રી નારણભાઇ રાઠવા દ્વારા ગુજરાત રાજ્યમાં વર્તમાનમાં કાર્યરત વન સ્ટોપ સેન્ટરની જિલ્લાવાર સ્થિતિ અંગે પ્રશ્ન પુછવામાં આવ્યો હતો.


    ( અ ) ગુજરાતમાં હાલમાં કાર્યરત ' વન સ્ટોપ સેન્ટર ' જિલ્લા વાર સંખ્યા તેમજ અત્યાર સુધીમાં ઉપરોક્ત ઉલ્લેખિત કેન્દ્રોમાં આવરી લેવામાં નથી આવ્યા તેવા જિલ્લાની માહિતી ; ( બ ) અત્યાર સુધીમાં બાકી રહેલા જિલ્લાઓમાં આવા કેન્દ્રો કાર્યરત ના કરવા પાછળનું કારણ ; અને ( ક ) છેલ્લા બે વર્ષમાં ' વન સ્ટોપ સેન્ટર્સ ' માટે ફાળવવામાં આવેલા અને તેની કામગીરી માટે ઉપયોગમાં લેવામાં આવેલા ભંડોળની વર્ષ અનુસાર માહિતી તેમજ ફાળવવામાં આવેલા ભંડોળમાંથી ઉપયોગમાં નથી લેવાઇ તેવી રકમની વિગતો અને તેનો ઉપયોગ ન કરવા પાછળના કારણની વિગતો.


    જવાબ માં મહિલા અને બાળવિકાસ મંત્રી શ્રીમતિ સ્મૃતિ ઝુબીન ઇરાનીએ જણાવ્યું હતું કે, “ ( અ ) અને ( બ ): ભારત સરકારના મહિલા અને બાળ વિકાસ મંત્રાલયે ગુજરાતના તમામ 33 જિલ્લામાં મંત્રાલયની વન સ્ટોપ સેન્ટર યોજના હેઠળ વન સ્ટોપ સેન્ટર ( OSC) શરૂ કરવાની મંજૂરી આપી છે. માન્યતા આપવામાં આવેલા OSC માંથી ગુજરાતની રાજ્ય સરકારે અત્યાર સુધીમાં 33 માંથી 32 જિલ્લામાં આવા કેન્દ્રો કાર્યરત કર્યા છે. ગુજરાતમાં કાર્યરત OSC ની વિગતવાર માહિતી બિડાણ – I માં આપવામાં આવી છે. ગુજરાતની રાજ્ય સરકાર દ્વારા આપવામાં આવેલી માહિતી અનુસાર , OSC શરૂ કરવા માટે અનુકૂળ જગ્યા અથવા સંકુલ શોધવામાં આવતી મુશ્કેલીના કારણે ગાંધીનગર જિલ્લામાં અત્યાર સુધી તેનો પ્રારંભ થઇ શક્યો નથી. છતા , ગાંધીનગરમાં OSC શરૂ કરવા અને તેને કાર્યરત કરવા માટે રાજ્ય સરકાર દ્વારા યોગ્ય પગલાં લેવામાં આવી રહ્યાં છે.


    ( ક ): ગુજરાતમાં છેલ્લા બે વર્ષમાં ' વન સ્ટોપ સેન્ટર ' ને ફાળવવામાં આવેલા ભંડોળ અને તેની એકંદરે કામગીરી પાછળ ઉપયોગમાં લેવાયેલી રકમની વર્ષ પ્રમાણે વિગતવાર માહિતી બિડાણ- II માં આપવામાં આવી છે. રાજ્ય અને જિલ્લા વહિવટી તંત્રો OSC સ્થાપવા અને તેને કાર્યરત કરવા માટે તેમજ OSC યોજના હેઠળ ભારત સરકાર દ્વારા ફાળવવામાં આવતા ભંડોળના ઉપયોગ માટે જવાબદાર છે. ”


    જવાબના સંદર્ભે જરૂરી બિડાણ – I અને II સાથે જોડેલ છે.
