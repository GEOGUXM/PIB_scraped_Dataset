      ପେଟ୍ରୋଲିୟମ ଓ ପ୍ରାକୃତିକ ବାଷ୍ପ ମନ୍ତ୍ରଣାଳୟ

      ପେଟ୍ରୋଲିୟମ୍‌ ଓ ପ୍ରାକୃତିକ ବାଷ୍ପ ମନ୍ତ୍ରଣାଳୟ ପକ୍ଷରୁ ଗ୍ରହଣ କରାଯାଇଥିବା କେତେକ ପ୍ରଶାସନିକ ସଂସ୍କାର
      Posted On: 07 JAN 2019 2:36PM by PIB Bhubaneshwar


    ଗତ ଚାରି ବର୍ଷ ମଧ୍ୟରେ ସରକାର ପେଟ୍ରୋଲିୟମ୍‌ ଓ ପ୍ରାକୃତିକ ବାଷ୍ପ ମନ୍ତ୍ରଣାଳୟର ଉଦ୍ଦେଶ୍ୟ ସାଧନ ସକାଶେ ଯେଉଁସବୁ ପ୍ରମୁଖ ନୀତିଗତ ପଦକ୍ଷେପ ଓ ସଂସ୍କାର କାର୍ଯ୍ୟକ୍ରମ ଆରମ୍ଭ କରିଛନ୍ତି ସେଗୁଡ଼ିକ ନିମ୍ନରେ ପ୍ରଦାନ କରାଗଲା: -

     1. ହାଇଡ଼୍ରୋକାର୍ବନ ଅନୁସନ୍ଧାନ ଲାଇସେନ୍ସ ପ୍ରଦାନ ନୀତି ଏବଂ ଓପନ ଏକରେଜନ ଲାଇସେନ୍ସ ପ୍ରଦାନ ନୀତିର ଚୂଡ଼ାନ୍ତକରଣ ;
     2. ଆବିଷ୍କୃତ କ୍ଷୁଦ୍ର କ୍ଷେତ୍ର ନୀତି ଚୂଡ଼ାନ୍ତକରଣ ;
     3. ସେଡ଼ିମେଣ୍ଟାରୀ ବେସିନ କ୍ଷେତ୍ରରେ ଅଣ ଆକଳିତ କ୍ଷେତ୍ରର ସର୍ଭେକରଣ ;
     4. ଜାତୀୟ ଡ଼ାଟା ରିପୋଜିଟୋରୀ ସ୍ଥାପନ ;
     5. ଅଣପାରମ୍ପରିକ ହାଇଡ଼୍ରୋକାର୍ବନ ଅନୁସନ୍ଧାନ ଏବଂ ଉତ୍ପାଦନ ସକାଶେ ନୀତିଗତ ଢ଼ାଞ୍ଚା ପ୍ରସ୍ତୁତକରଣ ;
     6. କୋଇଲା ଶଯ୍ୟା ମିଥେନର ତୁରନ୍ତ ମୂଲ୍ୟକରଣ ନୀତି ;
     7. ତୈଳ ରିକଭରୀ ପଦ୍ଧତିର ସଂପ୍ରସାରଣ ଓ ଉନ୍ନତିକରଣ ନୀତି ;
     8. ଗ୍ୟାସ ମୂଲ୍ୟ ନିର୍ଦ୍ଧାରଣ , ମାର୍କେଟିଂ ଓ ମୂଲ୍ୟାୟନ ସ୍ୱାଧୀନତାରେ ସଂସ୍କାରକରଣ
     9. ଉତ୍ପାଦନ ଚୁକ୍ତିନାମା ଓ ଆର୍ଥିକ ପ୍ରୋତ୍ସାହନ ପ୍ରଦାନ ବ୍ୟବସ୍ଥାର ସୁସଂଗତିକରଣ ;
    10. ଜାତୀୟ ଗ୍ୟାସ ଗ୍ରୀଡ଼ର ବିକାଶ ;
    11. ନଗରକ୍ଷେତ୍ର ଗ୍ୟାସ ବିତରଣ ନେଟୱାର୍କକୁ ଆଗକୁ ଆଗେଇ ନେବା ଦିଗରେ ପଦକ୍ଷେପ ;
    12. ଏଲଏନଜି/ ସିଏନଜିକୁ ପରିବହନ ସେକ୍ଟରେ ବ୍ୟବହାର ସକାଶେ ପ୍ରୋତ୍ସାହନ ପ୍ରଦାନ ;
    13. ପ୍ରଧାନମନ୍ତ୍ରୀ ଉଜ୍ଜ୍ୱଳା ଯୋଜନା ଜରିଆରେ ପିଜି ସଂଯୋଗ ଯୋଗାଣ ;
    14. ଏଲପିଜି ଉପଭୋକ୍ତାଙ୍କୁ ଛାଡ଼ି ଯେଉଁମାନଙ୍କର 10ଲକ୍ଷ ବା ତଦୁର୍ଦ୍ଧ୍ଵ ଟଙ୍କା ବାର୍ଷିକ ଉପାର୍ଜନ ହୋଇଥିବା ସେମାନଙ୍କ ସକାଶେ ସବସିଡ଼ି ଆଉଟ ଗୋ ବ୍ୟବସ୍ଥାର ସୁସଂଗତିକରଣ ;
      * xv. ପହଲ ମାଧ୍ୟମରେ ଏଲପିଜି ଉପଭୋକ୍ତାମାନଙ୍କୁ ସବସିଡ଼ି ରାଶି ଫେରସ୍ତ ବ୍ୟବସ୍ଥା ;
     1. 8ଟି ରାଜ୍ୟ/ କେନ୍ଦ୍ରଶାସିତ କ୍ଷେତ୍ରଙ୍କୁ କିରୋସିନମୁକ୍ତ ଘୋଷଣା ;
     2. କିରୋସିନ୍‌ ସାଧାରଣ ବଣ୍ଟନ ବ୍ୟବସ୍ଥାରେ ପ୍ରତ୍ୟକ୍ଷ ସୁବିଧା ହସ୍ତାନ୍ତରଣ ବ୍ୟବସ୍ଥା ଲାଗୁ ;
     3. ଡ଼ାଉନଷ୍ଟ୍ରମ୍‌ ଏବଂ ମିଡ଼ଷ୍ଟ୍ରୀମ୍‌ ଭିତ୍ତିଭୂମିରେ ବ୍ୟାପକ ଉନ୍ନତିକରଣ ;
     4. ରିଟେଲ ଆଉଟଲେଟଗୁଡ଼ିକର ଅଟୋମେସନ୍‌ ;
      * xx. ଡ଼ିଜିଟାଲ ପେମେଣ୍ଟ ବ୍ୟବସ୍ଥାକୁ ପ୍ରୋତ୍ସାହନ ପ୍ରଦାନ ;
     1. ଅଟୋ ଇନ୍ଧନ ନୀତି ପ୍ରବର୍ତ୍ତନ ;
     2. ସକ୍ରିୟତା ଓ ସ୍ୱଚ୍ଛତା ସକାଶେ ଇନ୍ଧନର ଦୈନିକ ମୂଲ୍ୟ ନିର୍ଦ୍ଧାରଣ ନୀତି ପ୍ରବର୍ତ୍ତନ ;
     3. ଇଜ୍‌ ଅଫ୍‌ ଲିଭିଂକୁ ପ୍ରୋତ୍ସାହନ ପ୍ରଦାନ ସକାଶେ ନାଗରିକକୈନ୍ଦ୍ରିକ ପଦକ୍ଷେପମାନ ଗ୍ରହଣ ;
     4. BS VI ଦିଗରେ ଧୀରେ ଧୀରେ ଅଗ୍ରସର ହେବା ;
     5. ହେଭି ଡ୍ୟୁଟି ଯାନବାହନ ପାଇଁ ଇନ୍ଧନ ନୀତି ବ୍ୟବସ୍ଥା ;
     6. ରାଷ୍ଟ୍ର ନିର୍ମାଣ ଦିଗରେ କର୍ପୋରେଟ ସାମାଜିକ ଦାୟିତ୍ୱବୋଧର ଉଚ୍ଚତର ମାନ ନିର୍ଦ୍ଧାରଣ ;
     7. ପେଟ୍ରୋଲରେ ଇଥାନୋଲ ମିଶ୍ରଣ କାର୍ଯ୍ୟକ୍ରମକୁ ପ୍ରୋତ୍ସାହନ ପ୍ରଦାନ ;
     8. ଜୈବ ଡ଼ିଜେଲ କାର୍ଯ୍ୟକ୍ରମକୁ ପ୍ରୋତ୍ସାହନ ;
     9. ଲିଗନୋସେଲୁଲୋଜେସ୍‌ ମାଧ୍ୟମରେ 2ଜି ଇଥାନୋଲକୁ ପ୍ରୋତ୍ସାହନ ପ୍ରଦାନ ;
    10. ହାଇଡ଼୍ରୋକାର୍ବନ ସେକ୍ଟରରେ କୌଶଳ ପରିଷଦ ସ୍ଥାପନ ;
    11. ଅଗ୍ରାଧିକାରଯୁକ୍ତ ସ୍ଥାନୀୟ ଦ୍ରବ୍ୟ କ୍ରୟ ନିମନ୍ତେ ନୀତି ବ୍ୟବସ୍ଥା ଗ୍ରହଣ ;
    12. ଅନେକ ରାଷ୍ଟ୍ରଙ୍କସହ ହାଇଡ଼୍ରୋକାର୍ବନ ସେକ୍ଟରକୁ କାର୍ଯ୍ୟକ୍ରମରେ ଯୋଡ଼ିବା ;
    13. ଏବଂ ଦରିଆପାରୀ ଅଧିଗ୍ରହଣ ସହ ହାଇଡ଼୍ରୋକାର୍ମନ ଫୁଟପ୍ରିଣ୍ଟର ସଂପ୍ରସାରଣ କରିବା ।

    ଉପର ବର୍ଣ୍ଣିତ ପଦକ୍ଷେପମାନ ଶକ୍ତି ଉପଲବ୍ଧତା , ଶକ୍ତି ସକ୍ରିୟତା , ଶକ୍ତି ପୋଷଣୀୟତା ଏବଂ ଶକ୍ତି ନିରାପତ୍ତାକୁ ସାକାର କରିବା ଦିଗରେ ବିଶେଷ ଭାବେ ସହାୟକ ହୋଇଛି ।


    କେନ୍ଦ୍ର ପେଟ୍ରୋଲିୟମ୍‌ ଓ ପ୍ରାକୃତିକ ବାଷ୍ପ ମନ୍ତ୍ରୀ ଶ୍ରୀ ଧର୍ମେନ୍ଦ୍ର ପ୍ରଧାନ ଲୋକସଭାରେ ଏକ ପ୍ରଶ୍ନର ଲିଖିତ ଉତ୍ତର ଦେଇ ଏହି ସୂଚନା ପ୍ରଦାନ କରିଛନ୍ତି ।
